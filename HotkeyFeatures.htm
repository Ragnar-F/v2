Erweiterte Hotkey-Features

Einige der am einfachsten zu erreichenden Tasten auf der Tastatur sind gleichzeitig auch solche, die am wenigsten genutzt werden. Weisen Sie diesen Tasten etwas Sinnvolles zu! Wenn Sie beispielsweise die rechte Alt-Taste nur selten nutzen, können Sie ihr eine Aktion zuweisen, die Sie häufig durchführen:

RAlt::
MsgBox "Sie haben die rechte ALT-Taste gedrückt."
return

Sie können das sogar tun, ohne auf die ursprüngliche Funktion der rechten Alt-Taste verzichten zu müssen, indem Sie die rechte Alt-Taste als "Präfix" für eine andere Taste bestimmen. Das folgende Beispiel zeigt, wie die rechte Alt-Taste zu einem Präfix gemacht werden kann, ohne dass die restlichen Tasten davon betroffen sind. Wenn Sie aber die rechte Alt-Taste drücken und loslassen, ohne mit ihr eine andere Taste modifiziert zu haben, wird ihre eigene Hotkey-Aktion (oben) ausgeführt:

RAlt & j::AltTab

 

STRG, ALT, UMSCHALT oder WIN sind nicht die einzigen möglichen Modifikatoren; es können zwei beliebige Tasten oder Maustasten kombiniert werden, um einen benutzerdefinierten Hotkey zu erstellen. Zum Beispiel: Halten Sie Numpad0 gedrückt und drücken Sie Numpad1, um einen Hotkey zu starten (Syntax: Numpad0 & Numpad1::); Halten Sie Feststell gedrückt und drücken Sie eine weitere Taste oder Maustaste (Syntax: CapsLock & RButton::). In diesem Fall bleibt der Status der Feststell-Taste (On oder Off) unverändert, wenn mit ihr der Hotkey ausgelöst wurde. Weitere Informationen finden Sie unter Benutzerdefinierte Tastenkombinationen.

 

Benutze das Mausrad (oder irgendeine andere Taste) als Ersatz für ALT+TAB. Drücken Sie das Mausrad, um das Alt-Tab-Fenster zu öffnen oder zu schließen, und drehen Sie das Mausrad, um in diesem Fenster zu navigieren. Wenn das Alt-Tab-Fenster geschlossen ist, funktioniert das Mausrad wie gewohnt. Syntax:

MButton::AltTabMenu
WheelDown::AltTab
WheelUp::ShiftAltTab

 

Wandeln Sie eine Taste in eine Maustaste um, oder lassen Sie eine Aktion kontinuierlich wiederholen, während Sie eine Taste oder Maustaste gedrückt halten. Weitere Beispiele finden Sie unter Neubelegung.

 

Machen Sie Ihre Hotkeys kontextabhängig: Benutzen Sie einfach zu erreichende Hotkeys, um je nach Fenstertyp eine Aktion auszulösen. Das folgende Beispiel zeigt, wie die rechte Strg-Taste unterschiedliche Aktionen durchführen kann, je nachdem, ob das aktive Fenster der Texteditor oder der Rechner ist:

#if WinActive("ahk_class Notepad")
RControl::MenuSelect ,, "Datei", "Speichern"  ; Speichert die aktuell geöffnete Datei im Texteditor.

#if WinActive("Rechner")
RControl::Send "^c!{tab}^v"  ; Kopiert das Ergebnis des Rechners in das zuvor aktive Fenster.

Weitere Informationen finden Sie unter #If.

 

Hotstrings: Definieren Sie Abkürzungen, die automatisch in lange Texte umgewandelt werden sollen. Es sind keine speziellen Schulungen oder Scripting-Kenntnisse erforderlich. Die folgenden Hotstrings beispielsweise wandeln die Abkürzungen in lange Texte um, sobald Sie usw, zb oder bzw geschrieben haben:

::usw::und so weiter
::zb::zum Beispiel
::bzw::beziehungsweise

(Mehr Details)

 

Hilfreiche Tipps für Spieler:

  • Reduzieren Sie die Abnutzung Ihrer Finger, indem Sie eine beliebige Taste zu einem Hotkey machen, inklusive einzelne Buchstaben, Pfeiltasten, Ziffernblocktasten, und sogar die Modifikatortasten (STRG/ALT/WIN/UMSCHALT).
  • Erstellen Sie Maus-Hotkeys, zum Beispiel für die mittlere Maustaste (MButton) oder zum Drehen oder Kippen des Mausrades (WheelUp, WheelDown, WheelLeft und WheelRight). Außerdem können Sie bei Bedarf die Maustasten mit den Tasten Ihrer Tastatur kombinieren. Damit ein Hotkey beispielsweise mit der Tastenkombination STRG+MAUSTASTE-RECHTS ausgelöst werden kann, ist ^RButton:: erforderlich.
  • Erstellen Sie "durchreichende" Hotkeys. Zum Beispiel kann die linke Maustaste sowohl eine Hotkey-Aktion auslösen als auch ihren normalen Klick an das Spiel senden (Syntax: ~LButton::).
  • Mit internen Funktionen wie PixelSearch, PixelGetColor und ImageSearch können Spielaktionen automatisiert werden.
  • Machen Sie von der Möglichkeit Gebrauch, Hotkeys mit dem Tastatur-Hook zu implementieren. Dieser Hook könnte die Reaktionsfähigkeit von Hotkeys erhöhen, wenn die CPU während eines Spiels stark belastet wird. Außerdem könnte der Hook die Restriktionen des Spiels überschreiben, eigene Tasten zum Auslösen von Spielaktionen zu bestimmen.

Weitere Informationen finden Sie im Abschnitt Hotkeys.
