Schaltflächennamen der MsgBox ändern

Das ist ein funktionierendes Beispiel-Skript, das einen Timer verwendet, um die Schaltflächennamen eines MsgBox-Dialogfensters zu ändern. Auch wenn die Schaltflächennamen geändert werden, erfordert der Rückgabewert der MsgBox immer noch, dass die Schaltflächen mit ihren ursprünglichen Namen angesteuert werden.

Dieses Skript herunterladen | Andere Beispiel-Skripte | Home

#SingleInstance
SetTimer "ChangeButtonNames", 50
Result := MsgBox("Button auswählen:", "Hinzufügen oder Entfernen", 4)
if Result = "Yes"
    MsgBox "Hinzufügen ausgewählt."
else
    MsgBox "Entfernen ausgewählt."

ChangeButtonNames()
{
    if !WinExist("Hinzufügen oder Entfernen")
        return  ; Warten.
    SetTimer , 0
    WinActivate
    ControlSetText "&Hinzufügen", "Button1"
    ControlSetText "&Entfernen", "Button2"
}
