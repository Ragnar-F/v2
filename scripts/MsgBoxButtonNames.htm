Button-Namen der MsgBox ändern

Das ist ein funktionierendes Beispiel-Skript, das einen Timer verwendet, um die Namen der Buttons in einem MsgBox-Dialogfenster zu ändern. Auch wenn die Button-Namen geändert werden, benötigt MsgBox's Rückgabewert weiterhin den ursprünglichen Namen des Buttons.

Dieses Skript herunterladen | Andere Beispiel-Skripte | Home

#SingleInstance
SetTimer "ChangeButtonNames", 50
Result := MsgBox("Button auswählen:", "Hinzufügen oder Entfernen", 4)
if Result = "Yes"
    MsgBox "Hinzufügen ausgewählt."
else
    MsgBox "Entfernen ausgewählt."
return

ChangeButtonNames()
{
    if !WinExist("Hinzufügen oder Entfernen")
        return  ; Warten.
    SetTimer , "Off"
    WinActivate
    ControlSetText "&Hinzufügen", "Button1"
    ControlSetText "&Entfernen", "Button2"
}
