ErrorLevel

Das ist eine interne Variable, die gesetzt wird, um den Erfolg oder Misserfolg einiger internen Funktionen zu kennzeichnen (nicht alle internen Funktionen ändern den Wert von ErrorLevel). Ein Wert von 0 zeigt für gewöhnlich Erfolg an, jeder andere Wert Misserfolg. ErrorLevel kann auch vom Benutzer selbst gesetzt werden.

Jeder Thread hat seinen eigenen ErrorLevel, der anfangs eine 0 enthält. Wenn der aktuelle Thread also von einem anderen unterbrochen wird, während der originale Thread wiederaufgenommen wird, enthält er weiterhin den originalen Wert von ErrorLevel, aber nicht den Wert, der vom unterbrochenen Thread gesetzt wurde.

ErrorLevel kann auf einen Wert beliebiger Art gesetzt werden (üblicherweise 0 oder 1).

Hinweis: Wenn der ErrorLevel einen Wert höher als 1 enthält, dann sollte ErrorLevel darauf überprüft werden, ob er einen Wert ungleich 0 enthält.
