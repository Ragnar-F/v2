SoundBeep

Erzeugt einen Ton aus dem PC-Lautsprecher.

SoundBeep Frequenz, Dauer

Parameter

Frequenz

Typ: Integer

Die Frequenz des Sounds. Es sollte eine Nummer zwischen 37 und 32767 sein. Lässt man diesen Parameter weg, wird standardmäßig eine Frequenz von 523 verwendet.

Dauer

Typ: Integer

Die Abspieldauer des Sounds in Millisekunden. Lässt man diesen Parameter weg, wird standardmäßig eine Dauer von 150 verwendet.

Bemerkungen

Das Skript wartet, bis der Sound wiedergegeben wurde, bevor es fortfährt. Darüber hinaus könnte die Reaktionsfähigkeit des Systems beim Erzeugen des Sounds reduziert sein.

Wenn der Computer keine Soundkarte hat, wird ein Standard-Piepton über den PC-Lautsprecher wiedergegeben.

Um vordefinierte Systemklänge anstelle von Pieptönen zu erzeugen, verwenden Sie den Sternchen-Modus von SoundPlay.

Beispiele

#1

SoundBeep  ; Abspielen der Standard-Tonhöhe und -dauer.
SoundBeep 750, 500  ; Abspielen einer höheren Tonhöhe für eine halbe Sekunde.
