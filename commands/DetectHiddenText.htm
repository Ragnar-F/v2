DetectHiddenText

Legt fest, ob unsichtbare Texte in einem Fenster entdeckt werden können. Das beeinflusst Befehle wie WinExist und WinActivate.

DetectHiddenText "On|Off"

Parameter

On|Off

On oder 1 (true): Versteckter Text wird erkannt. Das ist die Standardeinstellung.

Off oder 0 (false): Versteckter Text wird nicht erkannt.

Bemerkungen

"Versteckter Text" ist ein Begriff, der sich auf unsichtbare Steuerelemente eines Fensters bezieht. Deren Texte gelten somit als "versteckt". Das Ausschalten von DetectHiddenText ist beispielsweise hilfreich, wenn man den Unterschied zwischen den verschiedenen Bereichen eines Multi-Bereich- oder Multi-Tab-Fensters herausfinden will. Mithilfe von Windows Spy kann ermittelt werden, welcher Text vom aktiven Fenster versteckt ist. Diese Einstellung hat Einfluss auf alle Befehle, die einen Fenstertext-Parameter haben, z. B. WinActivate, WinActive, WinWait und WinExist.

Die Built-in-Variable A_DetectHiddenText enthält die aktuelle Einstellung (On oder Off).

Jeder neu gestartete Thread (z. B. ein Hotkey, ein benutzerdefinierter Menüpunkt oder eine zeitgesteuerte Subroutine) beginnt mit der Standardeinstellung des Befehls. Diese Standardeinstellung kann geändert werden, wenn man diesen Befehl bereits im automatischen Ausführungsbereich (am Anfang des Skripts) definiert.
