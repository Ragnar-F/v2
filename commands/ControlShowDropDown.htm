ControlShowDropDown

Macht die Auswahlliste einer ComboBox sichtbar.

ControlShowDropDown Steuerelement , Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Steuerelement

Typ: Zeichenkette, Integer oder Objekt

ClassNN-Bezeichnung, Text oder HWND-Nummer des Steuerelements, oder ein Objekt mit einer Hwnd-Eigenschaft. Einzelheiten finden Sie unter Der Steuerelement-Parameter.

Fenstertitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

Textausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt bei Erfolg eine 1 (true) und bei Misserfolg eine 0 (false) zurück.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Es wird eine Ausnahme ausgelöst, wenn ungültige Parameter erkannt werden.

Bemerkungen

Um die Zuverlässigkeit zu verbessern, wird nach jeder Benutzung dieser Funktion automatisch eine Verzögerung durchgeführt. Diese Verzögerung kann via SetControlDelay oder durch direktes Ändern des A_ControlDelay-Wertes geändert werden. Schauen Sie sich die Bemerkungen zu SetControlDelay an, um mehr zu erfahren.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

#1

Send "#r"  ; Zeigt das Ausführen-Dialogfenster an.
WinWaitActive "ahk_class #32770"  ; Wartet auf das Erscheinen des Dialogfensters.
ControlShowDropDown "ComboBox1"  ; Öffnet die DropDown-Liste. Der zweite Parameter wurde weggelassen, um das Zuletzt Gefundene Fenster zu verwenden.
Sleep 2000
ControlHideDropDown "ComboBox1"  ; Versteckt die DropDown-Liste.
Sleep 1000
Send "{Esc}"  ; Versteckt das Dialogfenster.
