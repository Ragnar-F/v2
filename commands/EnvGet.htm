EnvGet

Ruft eine Umgebungsvariable ab.

AusgabeVar := EnvGet(UmgebungsVar)

Parameter

AusgabeVar

Name der Variable, in der die Zeichenkette gespeichert werden soll.

UmgebungsVar

Der Name der Umgebungsvariable, die abgerufen werden soll. Zum Beispiel: AusgabeVar := EnvGet("Path").

Bemerkungen

Ist die angegebene Umgebungsvariable leer oder nicht vorhanden, wird AusgabeVar leer gemacht.

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB Text.

Diese Funktion existiert, weil normale Skriptvariablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Performance schlechter ist und zum anderen daran, dass Umgebungsvariablen auf 32 KB begrenzt sind.
