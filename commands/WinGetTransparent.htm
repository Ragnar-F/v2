WinGetTransparent

Gibt die Transparenzstärke eines bestimmten Fensters zurück.

Transparenz := WinGetTransparent(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Fenstertitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

Textausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Integer

Diese Funktion gibt einen Integer im Bereich von 0 bis 255 zurück, der die Transparenzstärke des angegebenen Fensters repräsentiert, wobei 0 ein durchsichtiges Fenster und 255 ein undurchsichtiges Fenster kennzeichnet (siehe WinSetTransparent, wie die Transparenz gesetzt werden kann). Der Rückgabewert ist eine leere Zeichenkette, wenn: 1) keine passenden Fenster gefunden werden; 2) das Fenster nicht transparent ist; oder 3) andere Bedingungen auftreten (durch das Verhalten des Betriebssystems), z. B. wenn das transparent gemachte Fenster minimiert, wiederhergestellt und/oder skaliert wurde.

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn das Fenster nicht gefunden werden konnte.

Bemerkungen

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

#1: Ruft die Transparenz eines Fensters unterhalb des Mauszeigers ab:
