{ ... } (Block) Blöcke sind Anweisungen, die von zwei geschweiften Klammern umschlossen sind. Diese werden häufig im Zusammenhang mit Funktionsdefinitionen und Ablaufsteuerungsanweisungen verwendet.

[ ... ] / Array Erstellt ein skriptfähiges assoziatives Array mit Integer-Keys.
