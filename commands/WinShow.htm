WinShow

Macht ein bestimmtes Fenster wieder sichtbar.

WinShow Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Fenstertitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

Textausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn das Fenster nicht gefunden werden konnte, es sei denn, der Gruppenmodus wird verwendet.

Bemerkungen

Standardmäßig ist WinShow die einzige Funktion, die in der Lage ist, versteckte Fenster immer zu erkennen. Andere interne Funktionen können dies nur, wenn DetectHiddenWindows eingeschaltet ist.

Diese Funktion agiert nur mit dem obersten übereinstimmenden Fenster, es sei denn, Fenstertitel ist ahk_group Gruppenname, dann sind alle Fenster in der Gruppe betroffen.

Siehe auch

WinHide, SetTitleMatchMode, DetectHiddenWindows, Zuletzt Gefundenes Fenster

Beispiele

#1

Run "notepad.exe"
WinWait "Unbenannt - Editor"
Sleep 500
WinHide  ; Durch Weglassen des Parameters wird das oben gefundene Fenster verwendet.
Sleep 1000
WinShow
