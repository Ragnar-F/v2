WinShow

Macht das angegebene Fenster wieder sichtbar.

WinShow Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Fenstertitel

Ein Fenstertitel oder sonstige Kriterien, die das Zielfenster ermitteln sollen. Siehe Fenstertitel.

Fenstertext

In diesem Parameter kann eine Zeichenkette angegeben werden, die in einem Textelement des Zielfensters vorkommt (abrufbar mit Windows Spy). Versteckte Textelemente werden auch erkannt, sofern DetectHiddenText aktiv ist.

Titelausnahme

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Standardmäßig ist WinShow die einzige Funktion, die in der Lage ist, versteckte Fenster immer zu erkennen. Andere interne Funktionen können dies nur, wenn DetectHiddenWindows eingeschaltet ist.

Diese Funktion bezieht sich immer auf das oberste übereinstimmende Fenster; enthält Fenstertitel allerdings ahk_group Gruppenname, sind alle Fenster in der Gruppe betroffen.

Siehe auch

WinHide, SetTitleMatchMode, DetectHiddenWindows, Zuletzt Gefundenes Fenster

Beispiel

Run "notepad.exe"
WinWait "Unbenannt - Editor"
Sleep 500
WinHide  ; Da hier der Parameter fehlt, wird das oben gefundene Fenster verwendet.
Sleep 1000
WinShow
