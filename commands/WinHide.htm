WinHide

Versteckt ein bestimmtes Fenster.

WinHide Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Mit WinShow kann ein verstecktes Fenster wieder sichtbar gemacht werden (DetectHiddenWindows kann entweder On oder Off sein, um das zu ermöglichen).

Diese Funktion bezieht sich immer auf das oberste übereinstimmende Fenster; enthält Fenstertitel allerdings ahk_group Gruppenname, sind alle Fenster in der Gruppe betroffen.

Die Explorer-Taskleiste kann wie folgt versteckt / sichtbar gemacht werden:

WinHide "ahk_class Shell_TrayWnd"
WinShow "ahk_class Shell_TrayWnd"

Siehe auch

WinShow, SetTitleMatchMode, DetectHiddenWindows, Zuletzt Gefundenes Fenster, Win-Funktionen

Beispiel

Run "notepad.exe"
WinWait "Unbenannt - Editor"
Sleep 500
WinHide ; zuletzt gefundenes Fenster verwenden
Sleep 1000
WinShow
