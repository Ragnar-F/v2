ComObjCreate

Erstellt ein COM-Objekt.

ComObject := ComObjCreate(CLSID , IID)

Parameter

CLSID

Typ: Zeichenkette

CLSID oder eine visuell lesbare ProgID des COM-Objekts.

IID

Typ: Zeichenkette

Ein Interface-Identifikator, der zurückgegeben werden soll. In den meisten Fällen wird dieser Parameter weggelassen; lässt man ihn weg, wird standardmäßig "{00020400-0000-0000-C000-000000000046}" (IID_IDispatch) verwendet.

Rückgabewert

Typ: ComObject

Diese Funktion gibt ein COM-Wrapper-Objekt zurück.

Wenn IID weggelassen wird oder das IDispatch-Interface identifiziert, ist das Wrapper-Objekt vom Typ VT_DISPATCH (9), und das Skript kann Eigenschaften und Methoden des Objekts unter Verwendung der normalen Objektsyntax aufrufen. Ansonsten ist das Wrapper-Objekt vom Typ VT_UNKNOWN (13). In beiden Fällen kann ComCall verwendet werden, um native Interface-Methoden aufzurufen.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst, z. B. wenn ein Parameter ungültig ist oder das Objekt nicht das via IID spezifizierte Interface unterstützt.

Beispiele

Eine ständig wachsende Liste von Beispielen finden Sie unter: https://www.autohotkey.com/forum/topic61509.html.

#1: Startet eine Instanz des Internet Explorers, macht ihn sichtbar und navigiert zu einer Webseite.

ie := ComObjCreate("InternetExplorer.Application")
ie.Visible := true  ; Funktioniert nicht korrekt unter IE7.
ie.Navigate("https://www.autohotkey.com/")

#2: Ermittelt den Pfad des aktuellen Desktophintergrunds (Wallpaper).

AD_GETWP_BMP := 0
AD_GETWP_LAST_APPLIED := 0x00000002
CLSID_ActiveDesktop := "{75048700-EF1F-11D0-9888-006097DEACF9}"
IID_IActiveDesktop := "{F490EB00-1240-11D1-9888-006097DEACF9}"
cchWallpaper := 260
GetWallpaper := 4

AD := ComObjCreate(CLSID_ActiveDesktop, IID_IActiveDesktop)
wszWallpaper := BufferAlloc(cchWallpaper * 2)
ComCall(GetWallpaper, AD, "ptr", wszWallpaper, "uint", cchWallpaper, "uint", AD_GETWP_LAST_APPLIED)
Wallpaper := StrGet(wszWallpaper, "UTF-16")
MsgBox "Wallpaper: " Wallpaper
