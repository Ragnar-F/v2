{...} (Block)

Zwei geschweifte Klammern kennzeichnen einen Block. Blöcke werden typischerweise bei Funktionen, Else, Loop, While-Schleifen und If-Anweisungen verwendet.

{
Beliebig viele Funktionen
}

Bemerkungen

Mit einem Block können zwei oder mehr Funktionen vereint werden. Es kann auch dazu verwendet werden, die Zugehörigkeiten von IF und ELSE zu ändern. Im nachfolgenden Beispiel wird das ELSE dem ersten statt dem zweiten IF zugeordnet:

if var1 = 1
{
    if var2 = "abc"
        sleep 1
}
else
    return

Blöcke sind praktisch überall nutzbar. Sinnvoll sind sie aber nur bei Funktionen, Else, Loop oder if-Anweisungen.

Wenn ein IF, ELSE, Loop, While-Schleife oder For-Schleife nur eine einzige Funktion hat, muss diese nicht in ein Block gesetzt werden. Allerdings kann das in bestimmten Fällen dabei helfen, das Skript besser lesbar oder verwaltbar zu machen.

Ein Block kann leer sein (ohne Funktionen), was nützlich sein könnte, wenn der Inhalt eines Blocks auskommentiert werden soll, ohne den Block selbst entfernen zu müssen.

One True Brace (OTB, K&R-Stil): Der OTB-Stil darf optional bei folgenden Anweisungen verwendet werden: if-Anweisungen, else, While-Schleife, For-Schleife, Loop, Funktionsdefinitionen, Try, und Catch. Dieser Stil erlaubt es, dass man die geschweifte Startklammer des Blocks direkt neben der Block-Anweisung setzen darf, anstatt einzeln auf der nachfolgenden Zeile. Zum Beispiel:

if x < y {
    ...
} else {
    ...
}
While x < y {
    ...
}
For k,v in obj {
    ...
}
Loop Wiederholungsanzahl {
    ...
}
MeineFunktion(x, y) {
    ...
}
Try {
    ...
} Catch e {
    ...
} Finally {
    ....
}

Ebenso kann eine Funktion oder eine andere Aktion auf der rechten Seite einer geschweiften Klammer vorkommen (außer die geschweifte Startklammer des OTB-Stils). Zum Beispiel:

if x = 1
{ MsgBox "Diese Zeile befindet sich auf der rechten Seite der geschweiften Startklammer. Sie wird jedes Mal ausgeführt, wenn die IF-Anweisung wahr ist."
    MsgBox "Das ist die nächste Zeile."
} MsgBox "Diese Zeile befindet sich auf der rechten Seite der geschweiften Endklammer. Sie wird bedingungslos ausgeführt."
