Integer

Konvertiert eine numerische Zeichenkette oder einen Gleitkommawert in einen Integer.

IntWert := Integer(Wert)

Rückgabewert

Das Ergebnis der Konvertierung von Wert in einen reinen Integer (mit dem Typnamen "Integer"), oder Wert selbst, wenn es bereits der richtige Typ ist.

Bemerkungen

Jegliche Nachkommastellen von Wert gehen verloren, was vergleichbar ist mit Wert < 0 ? Ceil(Wert) : Floor(Wert).

Wenn der Wert nicht konvertiert werden kann, wird eine Ausnahme ausgelöst.

Um festzustellen, ob ein Wert in einen Integer umgewandelt werden kann, nutzt man Wert is 'number'.
