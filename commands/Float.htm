Float

Konvertiert eine numerische Zeichenkette oder einen Integerwert in eine Gleitkommazahl.

GleitWert := Float(Wert)

Rückgabewert

Das Ergebnis der Konvertierung von Wert in eine reine Gleitkommazahl (mit dem Typnamen "Float"), oder Wert selbst, wenn es bereits der richtige Typ ist.

Bemerkungen

Wenn der Wert nicht konvertiert werden kann, wird eine Ausnahme ausgelöst.

Um festzustellen, ob ein Wert in eine Gleitkommazahl umgewandelt werden kann, nutzt man Wert is 'number'.
