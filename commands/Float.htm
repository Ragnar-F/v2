Float

Konvertiert eine numerische Zeichenkette oder einen Integerwert in eine Floating-Point-Zahl.

FltWert := Float(Wert)

Rückgabewert

Typ: Float

Diese Funktion gibt das Ergebnis der Konvertierung von Wert in eine reine Floating-Point-Zahl (mit dem Typnamen "Float"), oder Wert selbst, wenn es bereits der richtige Typ ist, zurück.

Bemerkungen

Wenn der Wert nicht konvertiert werden kann, wird eine Ausnahme ausgelöst.

Um festzustellen, ob ein Wert in eine Floating-Point-Zahl umgewandelt werden kann, nutzt man Wert is 'number'.
