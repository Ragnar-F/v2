WinWait

Wartet, bis ein bestimmtes Fenster existiert.

WinWait Fenstertitel, Fenstertext, Sekunden, Titelausnahme, Textausnahme

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertitel kann nur leer gelassen werden, wenn Fenstertext, Titelausnahme oder Textausnahme vorhanden ist.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Sekunden

Typ: Integer oder Float

Wie viele Sekunden gewartet werden sollen, bevor eine Zeitüberschreitung erfolgt und eine 0 zurückgegeben wird. Lassen Sie diesen Parameter weg oder leer, um unendlich lang zu warten. Die Angabe von 0 ist das gleiche wie, als würde man 0.5 angeben.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer

Diese Funktion gibt das HWND (eindeutige ID) eines passenden Fensters zurück, wenn eines gefunden wurde, oder 0, wenn die Funktion das Zeitlimit überschritten hat.

Bemerkungen

Wenn ein passendes Fenster gefunden wurde, wird die Funktion die restlichen Sekunden nicht abwarten. Stattdessen wird sie das Zuletzt Gefundene Fenster aktualisieren und einen Wert zurückgeben, damit das Skript die Ausführung fortsetzen kann.

Während die Funktion im Wartezustand ist, können neue Threads via Hotkeys, benutzerdefinierte Menüpunkte oder Timer gestartet werden.

Wenn ein anderer Thread die Inhalte der Variablen ändert, die als Parameter für diese Funktion verwenden wurden, wird die Funktion die Änderung nicht sehen -- sie wird weiter den Titel und Text verwenden, die ursprünglich in den Variablen vorhanden waren, als die Funktion anfing zu warten.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiel

Run "notepad.exe"
if WinWait("Unbenannt - Editor", , 3)
    WinMinimize  ; Minimiert das gefundene Fenster von WinWait.
else
    MsgBox "WinWait hat die Zeit überschritten."
