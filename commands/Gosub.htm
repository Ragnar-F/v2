Gosub

Springt zu einem bestimmten Label und setzt dort die Ausführung bis zu einem Return fort.

Gosub Label
Gosub("Label")

Parameter

Label

Typ: Zeichenkette

Name eines Labels, Hotkey-Labels oder Hotstring-Labels, das angesprungen werden soll, um alle Funktionen unterhalb von Label bis zu einem Return oder Exit aufzurufen. Ein Return bewirkt, dass das Skript zur ersten Funktion unterhalb von Gosub zurückspringt und dort die Ausführung fortsetzt. Exit beendet den aktuellen Thread.

Bemerkungen

Label kann nur eine Variable oder ein Ausdruck sein, wenn runde Klammern verwendet werden. Zum Beispiel würden Gosub MeineSub und Gosub("MeineSub") die MeineSub:-Subroutine ausführen.

Die Performance ist leicht reduziert, wenn man ein dynamisches Label benutzt (also eine Variable oder ein Ausdruck, der einen Labelnamen zurückgibt), weil das Ziellabel nicht nur einmal (beim Starten des Skripts), sondern jedesmal neu eingelesen werden muss. Es wird eine Fehlermeldung angezeigt, wenn das Label nicht existiert. Um das zu verhindern, ruft man vorher IsLabel auf. Zum Beispiel:

if IsLabel(VarMitLabelname)
    Gosub(VarMitLabelname)

Obwohl Gosub für einfache, allgemeine Subroutinen nützlich ist, wäre es ratsam, für komplexere Anliegen Funktionen zu verwenden.
