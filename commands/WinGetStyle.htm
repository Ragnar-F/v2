WinGetStyle / WinGetExStyle

Gibt den Style bzw. erweiterten Style eines bestimmten Fensters zurück.

Style := WinGetStyle(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)
ExStyle := WinGetExStyle(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer

Diese Funktionen geben entweder den Style oder erweiterten Style des angegebenen Fensters zurück. Wenn keine passenden Fenster gefunden werden, wird eine leere Zeichenkette zurückgegeben.

Bemerkungen

Die Style-Tabelle beinhaltet eine Auflistung einiger Styles.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

WinSetStyle / WinSetExStyle, ControlGetStyle / ControlGetExStyle, Styles-Tabelle, Win-Funktionen, Control-Funktionen

Beispiele

Beispiel #1

Stellt fest, ob ein Fenster den WS_DISABLED-Style hat:

Style := WinGetStyle("Mein Fenstertitel")
if (Style & 0x8000000)  ; 0x8000000 ist WS_DISABLED.
    MsgBox "Mit dem Fenster kann nicht interagiert werden."

Beispiel #2

Stellt fest, ob ein Fenster den WS_EX_TOPMOST-Style hat (immer im Vordergrund):

ExStyle := WinGetExStyle("Mein Fenstertitel")
if (ExStyle & 0x8)  ; 0x8 ist WS_EX_TOPMOST.
    MsgBox "Das Fenster ist immer im Vordergrund."
