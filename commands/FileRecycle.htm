FileRecycle

Verschiebt eine Datei oder ein Verzeichnis in den Papierkorb, falls möglich.

FileRecycle Dateimuster

Parameter

Dateimuster

Typ: Zeichenkette

Der Name einer einzelnen Datei oder eine musterbasierte Suche wie C:\Temp\*.tmp. Dateimuster wird im A_WorkingDir-Verzeichnis vermutet, wenn kein absoluter Pfad angegeben ist.

Um einen kompletten Ordner zu recyceln, lässt man den umgekehrten Schrägstrich nach seinem Namen weg.

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Mit SHFileOperation wird die eigentliche Arbeit getan. Diese Funktion kann eine Datei endgültig löschen, wenn sie nicht in den Papierkorb passt; es erscheint eine Warnung, bevor das geschieht.

Siehe auch

FileRecycleEmpty, FileDelete, FileCopy, FileMove

Beispiel

FileRecycle "C:\Temp-Dateien\*.tmp"
