IsSet

Gibt eine Nummer ungleich 0 zurück, wenn eine bestimmte Variable einen Wert zugewiesen bekommen hat.

IsSet := IsSet(Var)

Parameter

Var

Typ: Variable

Eine Referenz zu der Variable. Zum Beispiel: IsSet(MyVar).

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt 1 (true) zurück, wenn Var einen Wert zugewiesen bekommen hat, ansonsten 0 (false).

Bemerkungen

Eine Variable ohne zugewiesenen Wert wird auch als nicht initialisierte Variable bezeichnet.

Nicht initialisierte Variablen enthalten standardmäßig eine leere Zeichenkette, worauf einige Skript-Autoren spekulieren. Wenn allerdings #Warn UseUnset verwendet wurde, kann der Zugriff auf eine nicht initialisierte Variable mit fast allen Mitteln außer IsSet eine Warnmeldung auslösen. IsSet kann genutzt werden, um zu prüfen, ob eine Variable eine Initialisierung erfordert, z. B. um eine globale oder statische Variable bei der ersten Verwendung zu initialisieren.

Bekannte Einschränkung: Wenn der MsgBox-Warnmodus aktiv ist, hat das Auslösen der Warnmeldung für eine bestimmte Variable den Nebeneffekt, dass diese Variable initialisiert wird, um weitere Warnungen zu verhindern. Dies bewirkt, dass IsSet für nachfolgende Aufrufe True zurückgibt.
