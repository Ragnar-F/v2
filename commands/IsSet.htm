IsSet

Gibt eine Nummer ungleich 0 zurück, wenn eine bestimmte Variable einen Wert zugewiesen bekommen hat.

IsSet := IsSet(Var)

Parameter

Var

Typ: Variable

Eine Referenz zu der Variable. Zum Beispiel: IsSet(MyVar).

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt 1 (true) zurück, wenn Var einen Wert zugewiesen bekommen hat, ansonsten 0 (false).

Bemerkungen

Eine Variable ohne zugewiesenen Wert wird auch als nicht initialisierte Variable bezeichnet. Der Versuch, auf eine nicht initialisierte Variable zuzugreifen (mit fast allen anderen Mitteln außer dieser Funktion) führt dazu, dass standardmäßig eine Warnung angezeigt wird, aber in einer zukünftigen Version wird diese Warnung mit einem Fehler ersetzt.

IsSet kann genutzt werden, um zu prüfen, ob eine Variable eine Initialisierung erfordert, z. B. um eine globale oder statische Variable bei der ersten Verwendung zu initialisieren.

Hinweis: Statische Initialisierungen wie z. B. static mein_statisches_array := [] werden nur einmal ausgewertet, nämlich dann, wenn sie das erste Mal während der Ausführung des Skripts erreicht werden, so dass IsSet in der Regel nicht verwendet werden muss.

Bekannte Einschränkung: Wenn der MsgBox-Warnmodus aktiv ist, hat das Auslösen der Warnmeldung für eine bestimmte Variable den Nebeneffekt, dass diese Variable initialisiert wird, um weitere Warnungen zu verhindern. Dies bewirkt, dass IsSet für nachfolgende Aufrufe True zurückgibt.
