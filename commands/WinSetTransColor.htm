WinSetTransColor

Macht alle Pixel der ausgewählten Farbe innerhalb eines bestimmten Fensters unsichtbar.

WinSetTransColor Farbe , Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Farbe

Typ: Zeichenkette

Geben Sie einen Farbnamen oder RGB-Wert an (siehe Farbtabelle oder verwenden Sie PixelGetColor im RGB-Modus). Um den sichtbaren Bereich eines Fensters zusätzlich transparent zu machen, fügt man ein Leerzeichen (kein Komma) gefolgt von der Transparenzstufe (0-255) ein. Zum Beispiel: WinSetTransColor "EEAA99 150", Fenstertitel.

Geben Sie das Wort "Off" an (nicht-Groß-/Kleinschreibung-sensitiv), um die Transparenz eines Fensters komplett zu deaktivieren. Dies ist funktional identisch zu WinSetTransparent "Off", Fenstertitel. Das Wort "Off" unterscheidet sich von dem Wert 255, weil es eventuell die Performance verbessert und die Systemressourcennutzung verringert (aber wahrscheinlich nur in Windows XP oder wenn die Desktopgestaltung deaktiviert ist).

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem mitgelieferten Windows Spy Tool aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt bei Erfolg eine 1 (true) und bei Misserfolg eine 0 (false) zurück.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Ein Fehler tritt auf, wenn das Zielfenster nicht gefunden wird oder die Änderung nicht angewendet werden kann.

Bemerkungen

Dies macht es möglich, dass der Inhalt hinter dem Fenster durchscheint. Klickt der Benutzer auf einen unsichtbaren Pixel, wirkt sich der Klick auf das Fenster dahinter aus.

Um die bestehende TransColor-Einstellung eines Fensters zu ändern, ist es eventuell erforderlich, die Transparenz auszuschalten, bevor die Änderung vollzogen werden kann.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
