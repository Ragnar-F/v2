DirCreate

Erstellt ein(en) Verzeichnis/Ordner.

DirCreate VerzName

Parameter

VerzName

Name des Verzeichnisses, das sich im A_WorkingDir-Verzeichnis befindet, solange kein absoluter Pfad angegeben ist.

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

A_LastError wird auf das Ergebnis der Systemfunktion GetLastError() gesetzt.
