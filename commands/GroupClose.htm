GroupClose

Schließt das aktive Fenster, das gerade mit GroupActivate oder GroupDeactivate aktiviert wurde, und aktiviert das nächste Fenster in der Gruppe. Es kann auch alle Fenster einer Gruppe schließen.

GroupClose Gruppenname , Modus

Parameter

Gruppenname

Typ: Zeichenkette

Name der Gruppe, die ursprünglich mit GroupAdd definiert wurde.

Modus

Typ: Zeichenkette

Lässt man diesen Parameter weg, wird die Funktion das aktive Fenster schließen und das älteste Fenster in der Serie aktivieren. Um dieses Verhalten zu ändern, gibt man einen der folgenden Buchstaben an:

R: Das neueste (zuletzt aktive) Fenster wird aktiviert, aber nur, wenn beim Aufrufen der Funktion kein Fenster aus der Gruppe aktiv ist. "R" ist zum Beispiel nützlich, wenn Sie vorübergehend zu einer anderen Aufgabe wechseln müssen. Wenn Sie mit GroupActivate, GroupDeactivate oder GroupClose zur Gruppe zurückkehren, wird das Fenster, mit dem Sie zuletzt gearbeitet haben, aktiviert und nicht das älteste Fenster.

A: Alle Mitglieder der Gruppe werden geschlossen. Hat den gleichen Effekt wie WinClose("ahk_group Gruppenname").

Bemerkungen

Wenn der Modus-Parameter nicht der Buchstabe A ist, wird das Verhalten dieser Funktion dadurch bestimmt, ob die vorherige Aktion auf Gruppenname GroupActivate oder GroupDeactivate war. Wenn es GroupDeactivate wird diese Funktion das aktive Fenster nur schließen, wenn das Fenster kein Mitglied der Gruppe ist (ansonsten passiert nichts). Wenn es GroupActivate oder nichts war, wird dieser Befehl das aktive Fenster nur schließen, wenn das Fenster ein Mitglied der Gruppe ist (ansonsten passiert nichts). Dieses Verhalten erlaubt es, GroupClose einem Hotkey zuzuweisen, als Ergänzung zum Gruppenname's GroupActivate- oder GroupDeactivate-Hotkey.

Weitere Informationen zum Thema Fenstergruppen finden Sie unter GroupAdd.
