GuiCreate

Erstellt ein neues Gui-Objekt, das essentiell zum Erstellen und Verwalten einer GUI ist.

GuiObj := GuiCreate(Optionen, Titel := A_ScriptName, EreignisObj)

Parameter

Optionen

Typ: Zeichenkette

Dieser Parameter kann Optionen enthalten, die von Gui.Opt unterstützt werden.

Titel

Typ: Zeichenkette

Der Fenstertitel. Lässt man diesen Parameter weg, wird standardmäßig der aktuelle Wert von A_ScriptName verwendet.

EreignisObj

Typ: Objekt

Ein "Event-Sink" oder Objekt, mit dem Ereignisse verbunden werden sollen. Wenn EreignisObj angegeben ist, können OnEvent, OnNotify und OnCommand benutzt werden, um Methoden für EreignisObj zu registrieren, die beim Auslösen eines Ereignisses aufgerufen werden sollen. Wenn dieser Parameter weggelassen wird oder leer ist, wird jede Zeichenkette, die an OnEvent's Funktion-Parameter übergeben wird, als Funktionsname interpretiert.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu einem neuen Gui-Objekt zurück. Dieses Objekt bietet Methoden und Eigenschaften zum Erstellen und Verwalten von Fenstern, und Erstellen von Steuerelementen.
