ControlMove

Verschiebt ein Steuerelement oder ändert dessen Größe.

ControlMove X, Y, Breite, Höhe, Steuerelement, Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

X, Y

Typ: Integer

Die X- und Y-Koordinaten (in Pixel) der oberen linken Ecke der neuen Position des Steuerelements. Wenn einer der Koordinaten leer ist, wird die Position des Steuerelements in Bezug auf diese Koordinate nicht verändert. Die Koordinaten sind relativ zur oberen linken Ecke des Zielfenster-Clientbereichs; ControlGetPos kann genutzt werden, um diese Koordinaten zu ermitteln.

Breite, Höhe

Typ: Integer

Die neue Breite und Höhe des Steuerelements (in Pixel). Wenn eine der Abmessungen weggelassen wird oder leer ist, wird die Größe des Steuerelements in Bezug auf diese Abmessung nicht verändert.

Steuerelement

Typ: Zeichenkette, Integer oder Objekt

ClassNN, Text oder HWND des Steuerelements, oder ein Objekt mit einer Hwnd-Eigenschaft. Einzelheiten finden Sie unter Der Steuerelement-Parameter.

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Um die Zuverlässigkeit der Funktion zu verbessern, erfolgt jedes Mal, wenn sie durchgeführt wurde, automatisch eine Verzögerung. Diese Verzögerung kann via SetControlDelay geändert werden.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiel

SetTimer "ControlMoveTimer"
AusgabeVar := InputBox(, "Mein Eingabefenster")

ControlMoveTimer()
{
    if !WinExist("Mein Eingabefenster")
        return
    ; Ansonsten gilt das "zuletzt gefundene" Fenster von oben:
    SetTimer , "Off"
    WinActivate
    ControlMove 10,, 200,, "OK"  ; Verschiebt die OK-Schaltfläche nach links und macht sie breiter.
}
