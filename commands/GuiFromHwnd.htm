GuiFromHwnd

Ermittelt das Gui-Objekt eines GUI-Fensters, das zum angegebenen HWND gehört.

GuiObj := GuiFromHwnd(Hwnd , ÜbergeordnetesFenster := false)

Parameter

Hwnd
Das Window Handle (HWND) eines GUI-Fensters. So ein Fenster könnte via GuiCreate erstellt worden sein.
ÜbergeordnetesFenster
Wenn dieser Parameter True ist, wird ein GUI-Fenster, das dem angegebenen HWND am nächsten übergeordnet ist, automatisch gesucht und abgerufen.

Rückgabewert

Gibt ein Gui-Objekt zurück. Dieses Objekt bietet Methoden und Eigenschaften zum Erstellen und Verwalten von Fenstern, und Erstellen von Steuerelementen.

Bemerkungen

Zum Beispiel kann das HWND eines GUI-Fensters via Gui.Hwnd, WinExist oder WinGet abgerufen werden.

Beispiele

Gui := GuiCreate(, "Fenstertitel")
Gui.Add("Text",, "Beliebiger Text.")
Gui.Show()

MsgBox(GuiFromHwnd(Gui.Hwnd).Title)
