GuiFromHwnd

Ermittelt das Gui-Objekt eines GUI-Fensters, das zu einer bestimmten HWND-Nummer gehört.

GuiObj := GuiFromHwnd(Hwnd , ÜbergeordnetesFenster := false)

Parameter

Hwnd

Typ: Integer

Die HWND-Nummer (eindeutige ID) eines GUI-Fensters. So ein Fenster könnte via Gui() erstellt worden sein.

ÜbergeordnetesFenster

Typ: Integer (boolescher Wert)

Wenn dieser Parameter 1 (true) ist, wird ein GUI-Fenster, das der angegebenen HWND-Nummer am nächsten übergeordnet ist, automatisch gesucht und abgerufen.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu einem existierenden Gui-Objekt zurück. Dieses Objekt bietet Methoden und Eigenschaften zum Erstellen und Verwalten von Fenstern, und Erstellen von Steuerelementen.

Bemerkungen

Zum Beispiel kann die HWND-Nummer (eindeutige ID) eines GUI-Fensters via Gui.Hwnd, WinExist oder WinGetID abgerufen werden.

Beispiele

Ermittelt das Gui-Objekt über die HWND-Nummer des gerade erstellten GUI-Fensters und meldet dessen Titel.

MeineGui := Gui(, "Fenstertitel")
MeineGui.Add("Text",, "Beliebiger Text.")
MeineGui.Show()

MsgBox(GuiFromHwnd(MeineGui.Hwnd).Title)
