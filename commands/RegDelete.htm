RegDelete

Löscht einen Wert oder Subkey aus der Registry.

RegDelete KeyName, Wertname
RegDeleteKey KeyName

Parameter

KeyName

Der vollständige Name des Registry-Keys.

Der Name muss entweder mit HKEY_LOCAL_MACHINE, HKEY_USERS, HKEY_CURRENT_USER, HKEY_CLASSES_ROOT oder HKEY_CURRENT_CONFIG beginnen (oder jeweils die Abkürzung, wie z. B. HKLM). Um eine Remote-Registry anzusteuern, setzt man vor dem Rootkey den Computernamen und einen Schrägstrich. Zum Beispiel: \\workstation01\HKEY_LOCAL_MACHINE

KeyName kann nur weggelassen werden, wenn eine Registry-Schleife läuft - in diesem Fall gilt standardmäßig der Key des aktuellen Schleifenelements. Wenn das Element ein Subkey ist, wird standardmäßig der vollständige Name dieses Subkeys benutzt. Wenn das Element ein Wert ist, gilt für Wertname standardmäßig der Name dieses Wertes, was aber überschrieben werden kann.

Wertname

Der Name des Wertes, der gelöscht werden soll. Fehlt dieser Parameter ist er leer, wird der Standardwert des Keys gelöscht (außer wie oben erwähnt). Der Standardwert wird im RegEdit-Tool als "(Standard)" angezeigt.

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

A_LastError wird auf das Ergebnis der Systemfunktion GetLastError() gesetzt.

Bemerkungen

Warnung: Das Löschen von Registry-Einträgen kann erhebliche Probleme verursachen - also seien Sie vorsichtig!

Mit der Registry-Schleife können mehrere Registry-Keys und -Werte abgerufen und bearbeitet werden.

Innerhalb einer Registry-Schleife löschen RegDelete und RegDeleteKey nicht unbedingt das aktuelle Schleifenelement. Wenn das Element ein Subkey ist, wird RegDelete() nur seinen Standardwert löschen, während RegDeleteKey() den Key selbst löschen wird. Wenn das Element ein Wert ist, wird RegDeleteKey() den Key löschen, der diesen Wert enthält, inklusive alle Subkeys und Werte.

In den Bemerkungen zur Registry-Schleife finden Sie Hinweise, wie man auf die Registry eines Remotecomputers zugreifen kann.

Mit SetRegView können Einträge aus den 64-Bit-Registry-Bereichen in einem 32-Bit-Skript oder umgekehrt entfernt werden.
