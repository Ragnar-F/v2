WinGetCount

Gibt die Anzahl von vorhandenen Fenstern zurück, die bestimmte Kriterien erfüllen.

Anzahl := WinGetCount(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Anzahl der vorhandenen Fenster zurück, die den angegebenen Parametern entsprechen. Wenn keine passenden Fenster gefunden werden, wird 0 zurückgegeben.

Bemerkungen

Um die Anzahl aller Fenster im Betriebssystem zu ermitteln, lässt man alle vier Titel-/Text-Parameter weg.

Die ID des Fensters, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
