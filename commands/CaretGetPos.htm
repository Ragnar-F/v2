CaretGetPos

Gibt die aktuelle Position des Textcursors (Texteinfügemarke) zurück.

CaretGetPos(AusgabeX, AusgabeY)

Parameter

AusgabeX/Y

Namen der Variablen, in denen die X/Y-Koordinaten gespeichert werden sollen. Die ermittelten Koordinaten sind relativ zum Clientbereich des aktiven Fensters, sofern das nicht mit CoordMode oder A_CoordModeCaret überschrieben wurde.

Bemerkungen

Wenn kein Fenster aktiv ist oder die Position des Textcursors nicht ermittelt werden kann, werden die Ausgabevariablen leer gemacht. Die Funktion gibt True zurück, wenn das System eine Textcursor-Position zurückgegeben hat, was aber nicht unbedingt bedeutet, dass ein Textcursor sichtbar ist.

Man kann eine Ausgabevariable weglassen, wenn die entsprechende Information nicht benötigt wird.

Beachten Sie, dass einige Fenster (z. B. bestimmte Versionen von MS Word) immer die gleichen Positionsdaten melden, wenn Sie den Textcursor verschieben.

Siehe auch

CoordMode, A_CoordModeCaret

Beispiel

SetTimer "TextcursorAbfragen", 100
TextcursorAbfragen() {
    if CaretGetPos(x, y)
        ToolTip "X" x " Y" y, x, y - 20
    else
        ToolTip "Kein Textcursor vorhanden"
}
