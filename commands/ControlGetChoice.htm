ControlGetChoice

Gibt den Namen des aktuell ausgewählten ListBox- oder ComboBox-Listeneintrags zurück.

Auswahl := ControlGetChoice(Steuerelement , Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Steuerelement

Typ: Zeichenkette, Integer oder Objekt

ClassNN-Bezeichnung, Text oder HWND-Nummer des Steuerelements, oder ein Objekt mit einer Hwnd-Eigenschaft. Einzelheiten finden Sie unter Der Steuerelement-Parameter.

Fenstertitel

Typ: Zeichenkette, Integer oder Objekt

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Titeln enthalten, werden ignoriert.

Textausnahme

Typ: Zeichenkette

Alle Fenster, die diesen Wert in ihren Textelementen enthalten, werden ignoriert.

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt den Namen des aktuell ausgewählten ListBox- oder ComboBox-Listeneintrags zurück.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Es wird eine Ausnahme ausgelöst, wenn ungültige Parameter erkannt werden.

Bemerkungen

Das folgende Beispiel zeigt, wie man stattdessen die Positionsnummer des ausgewählten Listeneintrags abrufen kann (verwenden Sie aber nur eine der beiden Zeilen):

AuswahlPos := SendMessage(0x188, 0, 0, "ListBox1", Fenstertitel)  ; 0x188 ist LB_GETCURSEL (für eine ListBox).
AuswahlPos := SendMessage(0x147, 0, 0, "ComboBox1", Fenstertitel)  ; 0x147 ist CB_GETCURSEL (für eine DropDownList oder ComboBox).
AuswahlPos += 1  ; Wandelt 0-basiert in 1-basiert um, damit z. B. 1 dem ersten Eintrag entspricht.
; AuswahlPos ist nun 0, wenn kein Eintrag ausgewählt ist.

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
