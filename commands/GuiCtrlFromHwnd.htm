GuiCtrlFromHwnd

Ermittelt das GuiControl-Objekt eines GUI-Steuerelements, das zu einem bestimmten HWND gehört.

GuiControlObj := GuiCtrlFromHwnd(Hwnd)

Parameter

Hwnd
Das Window Handle (HWND) eines GUI-Steuerelements. So ein Steuerelement könnte via Gui.Add erstellt worden sein.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu einem existierenden GuiControl-Objekt zurück. Dieses Objekt hat Methoden und Eigenschaften zum Modifizieren von GUI-Steuerelementen und Abrufen ihrer Informationen.

Bemerkungen

Zum Beispiel kann das HWND eines GUI-Steuerelements via GuiCtrl.Hwnd, MouseGetPos oder OnMessage abgerufen werden.
