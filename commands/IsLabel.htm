IsLabel

Gibt eine Nummer ungleich 0 zurück, wenn ein bestimmtes Label im Skript vorkommt.

IstLabel := IsLabel(Labelname)

Parameter

Labelname

Typ: Zeichenkette

Der Name eines Labels.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt 1 (true) zurück, wenn das angegebene Label existiert, oder 0 (false), wenn nicht.

Bemerkungen

Diese Funktion ist nützlich, um Laufzeitfehler beim Spezifizieren eines dynamischen Labels für Gosub oder Goto zu vermeiden.

Beispiele

#1

if IsLabel("Label")
    MsgBox "Subroutine existiert"
else
    MsgBox "Subroutine existiert nicht"
