EnvSet

Schreibt einen Wert in eine Umgebungsvariable.

EnvSet UmgebungsVar, Wert

Parameter

UmgebungsVar
Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".
Wert

Ein Wert, der in die Umgebungsvariable gespeichert werden soll.

Rückgabewert

Typ: Integer (logisch)

Diese Funktion gibt bei Erfolg eine 1 (true) und bei Misserfolg eine 0 (false) zurück.

Bemerkungen

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB Text.

Eine Umgebungsvariable, die mit dieser Funktion erstellt oder geändert wurde, ist nur für Programme zugänglich, die das Skript via Run oder RunWait gestartet hat. Weitere Informationen finden Sie unter Umgebungsvariablen.

Diese Funktion existiert, weil normale Skriptvariablen nicht in die Systemumgebung gespeichert werden. Das liegt zum einen daran, dass die Performance schlechter ist und zum anderen daran, dass Umgebungsvariablen auf 32 KB begrenzt sind.

Siehe auch

EnvGet, Umgebungsvariablen, Run, RunWait

Beispiel

EnvSet "AutGUI", "Beliebiger Text für die Variable."
