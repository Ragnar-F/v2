EnvSet

Schreibt einen Wert in eine Umgebungsvariable.

EnvSet UmgebungsVar, Wert

Parameter

UmgebungsVar
Name der Umgebungsvariable, z. B. "COMSPEC" oder "PATH".
Wert

Wert, der in die Umgebungsvariable gespeichert werden soll.

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Bemerkungen

Das Betriebssystem limitiert jede Umgebungsvariable auf 32 KB.

Umgebungsvariablen, die mit dieser Funktion erstellt oder geändert wurden, können nur bei Programmen verwendet werden, die ein Skript per Run oder RunWait gestartet hat. Siehe Umgebungsvariablen für mehr Details.

Diese Funktion existiert, weil normale Skriptvariablen nicht in die Systemumgebung gespeichert werden. Das Speichern eines Wertes in Systemvariable hätte zum einen eine geringere Performance und zum anderen eine Begrenzung von 32 KB.

Siehe auch

EnvGet, Umgebungsvariablen, Run, RunWait

Beispiel

EnvSet "AutGUI", "Beliebiger Text für die Variable."
