IsByRef()

Gibt eine Nummer ungleich 0 zurück, wenn ein ByRef-Parameter einer Funktion mit einer Variable übergeben wurde.

TrueOderFalse := IsByRef(VarName)

Parameter

VarName

Name der Variable (ohne Anführungszeichen). Zum Beispiel: IsByRef(MeineVar).

Rückgabewert

Diese Funktion gibt eine 1 zurück, falls VarName ein ByRef-Parameter ist und der Aufrufer eine Variable übergeben hat; oder 0, falls VarName ein anderer Variablentyp ist.

MsgBox, % Funktion(MeineVar)

Funktion(ByRef Param)
{
    return IsByRef(Param)
}
