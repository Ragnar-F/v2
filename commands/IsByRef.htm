IsByRef

Gibt eine Nummer ungleich 0 zurück, wenn ein ByRef-Parameter einer Funktion mit einer bestimmten Variable belegt wurde.

IstByRef := IsByRef(ParameterVar)

Parameter

ParameterVar

Eine Referenz zu der Variable. Zum Beispiel: IsByRef(MeinParameter).

Rückgabewert

Typ: Integer (logisch)

Diese Funktion gibt 1 (true) zurück, falls ParameterVar ein ByRef-Parameter ist und der Aufrufer eine Variable übergeben hat; oder 0 (false), falls ParameterVar ein anderer Variablentyp ist.
