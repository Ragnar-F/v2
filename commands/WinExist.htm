WinExist

Überprüft, ob ein bestimmtes Fenster existiert.

EindeutigeID := WinExist(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

EindeutigeID

Typ: Integer

Eindeutige ID (HWND) des ersten Fensters, das mit den angegebenen Kriterien übereinstimmt.

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem mitgelieferten Windows Spy Tool aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Eindeutige ID (HWND) des ersten übereinstimmenden Fensters zurück (oder 0, wenn keine gefunden werden).

Da alle Zahlen ungleich Null als "wahr" angesehen werden, ist die Anweisung if WinExist(Fenstertitel) immer dann wahr, wenn Fenstertitel vorhanden ist.

Bemerkungen

Wenn alle Parameter fehlen, wird das Zuletzt Gefundene Fenster überprüft, ob es noch vorhanden ist.

Wenn ein geeignetes Fenster vorhanden ist, wird das Zuletzt Gefundene Fenster aktualisiert, um dieses Fenster zu sein.

Um das HWND eines Steuerelements zu ermitteln (z. B. für PostMessage, SendMessage oder DllCall), nutzt man ControlGetHwnd oder MouseGetPos.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Beispiele

; Beispiel 1
if WinExist("Unbenannt - Editor")
{
    WinActivate  ; Verwendet automatisch das oben gefundene Fenster.
    WinMaximize  ; das gleiche
    Send "Dies ist ein Text.{Enter}"
    return
}

; Beispiel 2
if !WinExist("Rechner")
    return
else
{
    WinActivate  ; Das obige "WinExist" setzt auch das "zuletzt gefundene" Fenster für uns.
    WinMove 40, 40  ; Verschiebt es auf eine neue Position.
    return
}

; Beispiel 3
if WinExist("ahk_class Notepad") or WinExist("ahk_class" . Klassenname)
    WinActivate  ; Verwendet das zuletzt gefundene Fenster.

MsgBox "Die ID des aktiven Fensters ist " . WinExist("A")
