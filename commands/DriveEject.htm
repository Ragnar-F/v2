DriveEject

Öffnet oder schließt ein bestimmtes CD- oder DVD-Laufwerk.

DriveEject Laufwerk, Schließen := false

Parameter

Laufwerk

Typ: Zeichenkette

Der Laufwerksbuchstabe, gefolgt von einem Doppelpunkt und einem optionalen umgekehrten Schrägstrich (UNC-Netzwerkpfade und zugeordnete Laufwerke wären auch möglich). Lässt man diesen Parameter weg, wird das Standard-CD/DVD-Laufwerk verwendet.

Schließen

Typ: Integer (boolescher Wert)

Geben Sie 1 (true) an, um das Laufwerk zu schließen. Geben Sie 0 (false) an oder lassen Sie diesen Parameter weg, um das Laufwerk zu öffnen.

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst, sofern dieser erkannt wurde.

Bemerkungen

Während diese Funktion das Laufwerk ein- oder ausfährt, wartet das Skript.

Diese Funktion wird eventuell nicht funktionieren, wenn das angegebene Laufwerk ein Netzwerk- oder Nicht-CD/DVD-Laufwerk ist. Wenn es in solchen Fällen oder aus einem anderen Grund fehlschlägt, wird eine Ausnahme ausgelöst. Um andere Medien- oder Gerätetypen auszufahren, siehe das DllCall-Beispiel ganz unten.

Es ist möglich, den vorherigen Zustand des Laufwerks zu ermitteln. Messen Sie einfach die Zeit, die die Funktion zum Ausführen benötigt. Der folgende Hotkey beispielsweise schaltet das Laufwerk jeweils in den entgegengesetzten Zustand (offen oder geschlossen):

#c::
{
    DriveEject()
    ; Wenn die Funktion schnell ausgeführt wurde, war das Laufwerk wahrscheinlich schon ausgefahren.
    ; In diesem Fall kann es eingefahren werden:
    if (A_TimeSinceThisHotkey < 1000)  ; Passen Sie diese Zeit an, falls notwendig.
        DriveEject(, true)
}

Beispiele

#1: Schließt das Standard-CD/DVD-Laufwerk:

DriveEject(, true)

#2: Dieses Beispiel ist eine alternative Methode, um neben CD/DVD noch andere Typen von Medien/Geräten ausfahren zu können. Fügen Sie in die erste Zeile den gewünschten Laufwerksbuchstaben ein (alle anderen Zeilen können ignoriert werden):

Laufwerksbuchstabe := "I:"  ; Tragen Sie hier das Laufwerk ein, das ausgefahren werden soll.

hVolume := DllCall("CreateFile"
    , "Str", "\\.\" . Laufwerksbuchstabe
    , "UInt", 0x80000000 | 0x40000000  ; GENERIC_READ | GENERIC_WRITE
    , "UInt", 0x1 | 0x2  ; FILE_SHARE_READ | FILE_SHARE_WRITE
    , "UInt", 0
    , "UInt", 0x3  ; OPEN_EXISTING
    , "UInt", 0, "UInt", 0)
if (hVolume != -1)
{
    DllCall("DeviceIoControl"
        , "UInt", hVolume
        , "UInt", 0x2D4808   ; IOCTL_STORAGE_EJECT_MEDIA
        , "UInt", 0, "UInt", 0, "UInt", 0, "UInt", 0
        , "UIntP", dwBytesReturned  ; Unbenutzt.
        , "UInt", 0)
    DllCall("CloseHandle", "UInt", hVolume)
}
