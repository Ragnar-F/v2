Deref

Erweitert Variablenverweise und Escapesequenzen, die sich in anderen Variablen befinden.

Deref, AusgabeVar, Zeichenkette

Parameter

AusgabeVar

Name der Variable, in der das Ergebnis gespeichert werden soll.

Zeichenkette

Eine Zeichenkette mit Variablenverweisen und/oder Escapesequenzen.

Bemerkungen

Jeder schlecht formatierte Variablenverweis wird im erweiterten Ergebnis weggelassen. Das gleiche gilt auch, wenn sich AusgabeVar in sich selbst erweitert. Das heißt, dass Verweise auf AusgabeVar innerhalb der Variablen von Zeichenkette beim Erweitern weggelassen werden (allerdings kann Zeichenkette selbst %AusgabeVar% sein).

Innerhalb einer Funktion wird jeder Variablenverweis in der Zeichenkette immer in eine lokale Variable aufgelöst. Die Standard-Regeln bzgl. der Gültigkeitsbereiche von Variablen gelten hier auch.

Beispiele

var1 := "test"
var2 := Chr(37) "var1" Chr(37)  ; Chr(37) ist %
Deref AusgabeVar, %var2%
MsgBox %AusgabeVar%  ; Zeigt "test" an
