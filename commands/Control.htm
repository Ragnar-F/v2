Control-Funktionen

Funktionen zum Abrufen von Informationen über ein Steuerelement, oder zum Modifizieren eines Steuerelements. Klicken Sie auf einen Funktionsnamen, um mehr zu erfahren.

Funktion Beschreibung
ControlAddItem Fügt eine bestimmte Zeichenkette als neuen Eintrag an das untere Ende einer ListBox oder ComboBox hinzu.
ControlChoose Setzt die Auswahl in einer ListBox oder ComboBox auf die angegebene Eintragsnummer.
ControlChooseString Setzt die Auswahl in einer ListBox oder ComboBox auf den ersten Eintrag, der mit der angegebenen Zeichenkette beginnt.
ControlClick Sendet einen Maustastenklick oder eine Mausradaktion an ein Steuerelement.
ControlDeleteItem Entfernt die angegebene Eintragsnummer aus einer ListBox oder ComboBox.
ControlEditPaste Fügt eine bestimmte Zeichenkette an dem Textcursor (Texteinfügemarke) eines Edit-Steuerelements ein.
ControlFindItem Gibt die Positionsnummer eines ListBox- oder ComboBox-Listeneintrags zurück, der exakt mit einer bestimmten Zeichenkette übereinstimmt.
ControlFocus Setzt den Eingabefokus auf das angegebene Steuerelement innerhalb eines Fensters.
ControlGetChecked Gibt einen Wert ungleich 0 zurück, wenn ein CheckBox- oder Radio-Steuerelement markiert ist.
ControlGetChoice Gibt den Namen des derzeit ausgewählten ListBox- oder ComboBox-Listeneintrags zurück.
ControlGetCurrentCol Gibt die Positionsnummer des Edit-Steuerelement-Textcursors auf horizontaler Ebene (Spaltennummer) zurück.
ControlGetCurrentLine Gibt die Positionsnummer des Edit-Steuerelement-Textcursors auf vertikaler Ebene (Zeilennummer) zurück.
ControlGetEnabled Gibt einen Wert ungleich 0 zurück, wenn die Interaktion mit einem bestimmten Steuerelement möglich ist.
ControlGetFocus Ermittelt, welches Steuerelement des Zielfensters den Eingabefokus hat, falls vorhanden.
ControlGetHwnd Gibt die eindeutige ID-Nummer eines bestimmten Steuerelements zurück.
ControlGetLine Gibt den Text einer bestimmten Zeile in einem Edit-Steuerelement zurück.
ControlGetLineCount Gibt die Anzahl der Zeilen eines Edit-Steuerelements zurück.
ControlGetList Gibt eine Liste von Einträgen/Reihen aus einer ListView, ListBox, ComboBox oder DropDownList zurück.
ControlGetPos Ruft die Position und Größe eines Steuerelements ab.
ControlGetSelected Gibt den markierten Text eines Edit-Steuerelements zurück.
ControlGetStyle
ControlGetExStyle		 Gibt einen Integer zurück, der den Style oder erweiterten Style eines bestimmten Steuerelements repräsentiert.
ControlGetTab Gibt die Positionsnummer des ausgewählten Tabs eines SysTabControl32-Steuerelements zurück.
ControlGetText Ruft den Text eines Steuerelements ab.
ControlGetVisible Gibt einen Wert ungleich 0 zurück, wenn ein bestimmtes Steuerelement sichtbar ist.
ControlHide Versteckt ein bestimmtes Steuerelement.
ControlHideDropDown Versteckt die Auswahlliste eines ComboBox-Steuerelements.
ControlMove Verschiebt ein Steuerelement oder ändert dessen Größe.
ControlSend
ControlSendText		 Sendet künstliche Tastatureingaben an ein Fenster oder Steuerelement.
ControlSetChecked Setzt oder entfernt die Markierung eines CheckBox- oder Radio-Steuerelements.
ControlSetEnabled Ermöglicht oder sperrt die Interaktion mit einem bestimmten Steuerelement.
ControlSetStyle
ControlSetExStyle		 Ändert den Style oder erweiterten Style eines bestimmten Steuerelements.
ControlSetTab Wählt anhand einer bestimmten Positionsnummer einen Tab eines SysTabControl32-Steuerelements aus.
ControlSetText Ändert den Text eines Steuerelements.
ControlShow Macht ein bestimmtes Steuerelement, das zuvor versteckt wurde, wieder sichtbar.
ControlShowDropDown Zeigt die Auswahlliste eines ComboBox-Steuerelements an.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt. Zusätzlich wird, sofern nicht anders angegeben, jede Funktion bei Erfolg eine 1 (True) und bei Misserfolg eine 0 (False) zurückgeben.

Es wird eine Ausnahme ausgelöst, wenn ungültige Parameter erkannt werden.

Bemerkungen

Um die Zuverlässigkeit zu verbessern, erfolgt nach jeder Verwendung einer steuerelement-modifizierenden Funktion automatisch eine Verzögerung (außer bei ControlSetStyle und ControlSetExStyle). Diese Verzögerung kann via SetControlDelay oder durch direktes Ändern des A_ControlDelay-Wertes geändert werden. Schauen Sie sich die Bemerkungen zu SetControlDelay an, um mehr zu erfahren.

MouseGetPos kann genutzt werden, um das ClassNN oder HWND des Steuerelements unterhalb des Mauszeigers zu ermitteln.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
