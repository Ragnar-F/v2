SoundSetVolume

Ändert eine Lautstärkeeinstellung eines Audiogeräts.

SoundSetVolume Einstellung , Komponente, Gerät

Parameter

Einstellung

Typ: Zahl oder Zeichenkette

Eine Prozentzahl zwischen -100 und 100 (kann eine Floating-Point-Zahl sein). Wenn die Zahl mit einem Plus- oder Minuszeichen beginnt, wird die aktuelle Einstellung mit der angegebenen Menge nach oben oder nach unten reguliert. Andernfalls wird die Einstellung explizit gesetzt.

Komponente

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer der Komponente. Zum Beispiel 1, "Line in" oder "Line in:2".

Wenn dieser Parameter weggelassen wird oder leer ist, wird die Master-Lautstärkeeinstellung geändert.

Weitere Informationen finden Sie unter Komponente (Sound-Funktionen).

Gerät

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer des Geräts. Zum Beispiel 1, "Lautsprecher", "Lautsprecher:2" oder "Lautsprecher (Beispiel HD Audio)".

Lässt man diesen Parameter weg, wird standardmäßig das als Standard festgelegte Wiedergabegerät des Systems verwendet (was nicht unbedingt Gerät 1 ist).

Weitere Informationen finden Sie unter Gerät (Sound-Funktionen).

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst. Die Message-Eigenschaft des Exception-Objekts kann eine der folgenden Zeichenketten enthalten:

  • Device not found (Gerät nicht gefunden)
  • Component not found (Komponente nicht gefunden)
  • Component doesn't support this control type (Komponente unterstützt diesen Regler nicht)
  • Sonstiges: Ein interner Funktionsaufruf ist fehlgeschlagen.

Bemerkungen

Zum Anpassen der Master-Lautstärke des Systems kann alternativ auch ein Skript verwendet werden, das lautstärkespezifische Tastendrücke sendet. Zum Beispiel:

Send "{Volume_Up}"  ; Erhöht die Master-Lautstärke um 1 Stufe (normalerweise 5%).
Send "{Volume_Down 3}"  ; Verringert die Master-Lautstärke um 3 Stufen.

Um den Leistungsumfang der installierten Audiogeräte auf dem Betriebssystem zu ermitteln -- z. B. die Namen und verfügbaren Komponenten -- können Sie das Soundkartenanalyse-Skript ausführen.

SoundSetVolume versucht, die vorhandene Balance beim Ändern der Lautstärke beizubehalten.

Mit SoundGetVolume kann die aktuelle Lautstärkeeinstellung abgerufen werden.

Beispiele

#1: Einfache Beispiele

SoundSetVolume 50  ; Setzt die Master-Lautstärke auf 50%
SoundSetVolume "+10"  ; Erhöht die Master-Lautstärke um 10%
SoundSetVolume -10  ; Verringert die Master-Lautstärke um 10%
SoundSetVolume "+20", , "Mikrofon"  ; Erhöht die Mikrofonaufnahmelautstärke um 20%
