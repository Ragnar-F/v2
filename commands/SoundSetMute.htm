SoundSetMute

Ändert die Stummschalteinstellung eines Audiogeräts.

SoundSetMute Einstellung , Komponente, Gerät

Parameter

Einstellung

Typ: Zahl oder Zeichenkette

Eine beliebige positive Zahl, um die Einstellung einzuschalten (stummzuschalten), oder eine 0, um sie auszuschalten (Stummschaltung aufzuheben). Wenn die Zahl allerdings mit einem Plus- oder Minuszeichen beginnt, wird die Einstellung umgeschaltet (jeweils auf dem entgegengesetzten Zustand).

Komponente

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer der Komponente. Zum Beispiel 1, "Line in" oder "Line in:2".

If omitted or blank, xxx.

Weitere Informationen finden Sie unter Komponente (Sound-Funktionen).

Gerät

Typ: Zeichenkette oder Integer

Anzeigename und/oder Indexnummer des Geräts. Zum Beispiel 1, "Lautsprecher", "Lautsprecher:2" oder "Lautsprecher (Beispiel HD Audio)".

Lässt man diesen Parameter weg, wird standardmäßig das als Standard festgelegte Wiedergabegerät des Systems verwendet (was nicht unbedingt Gerät 1 ist).

Weitere Informationen finden Sie unter Gerät (Sound-Funktionen).

Fehlerbehandlung

ErrorLevel wird auf 0 gesetzt, wenn die Funktion erfolgreich war. Ansonsten wird es auf einen der folgenden Sätze gesetzt:

  • Device Not Found (Gerät nicht gefunden)
  • Component Not Found (Komponente nicht gefunden)
  • Component Doesn't Support This Control Type (die Komponente unterstützt nicht diesen Kontrolltyp)
  • Can't Get Current Setting (aktuelle Einstellung konnte nicht ermittelt werden)
  • Can't Change Setting (aktuelle Einstellung konnte nicht geändert werden)

Bemerkungen

Zum Umschalten der Master-Stummschalteinstellung des als Standard festgelegten Wiedergabegeräts kann alternativ auch ein Skript verwendet werden, das Tastendrücke sendet. Zum Beispiel:

Send "{Volume_Mute}"  ; Gesamtlautstärke stumm schalten bzw. Stummschaltung aufheben.

Um den Leistungsumfang der installierten Audiogeräte auf dem Betriebssystem zu ermitteln -- z. B. die Namen und verfügbaren Komponenten -- können Sie das Soundkartenanalyse-Skript ausführen.

Mit SoundGetMute kann der aktuelle Wert einer Einstellung abgerufen werden.

Beispiele

#1: Einfache Beispiele

SoundSetMute true  ; Master-Lautstärke stummschalten
SoundSetMute -1  ; Schaltet die Stummschaltung um (jeweils auf den entgegengesetzten Zustand)
SoundSetMute 1, "Line In"  ; Line-In stummschalten
SoundSetMute 1, , "Mikrofon"  ; Mikrofonaufnahme stummschalten
