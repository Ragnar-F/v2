Trim

Entfernt Zeichen vom Anfang und/oder Ende einer Zeichenkette.

Ergebnis :=  Trim(Zeichenkette , ZuEntfernendeZeichen := " `t")
Ergebnis := LTrim(Zeichenkette , ZuEntfernendeZeichen := " `t")
Ergebnis := RTrim(Zeichenkette , ZuEntfernendeZeichen := " `t")

Parameter

Zeichenkette

Typ: Zeichenkette

Beliebige Zeichenkette oder Variable. Zahlen werden nicht unterstützt.

ZuEntfernendeZeichen

Typ: Zeichenkette

Eine optionale Liste von Zeichen (Groß-/Kleinschreibung-sensitiv), die am Anfang und Ende der Zeichenkette entfernt werden sollen. Lässt man diesen Parameter weg, werden standardmäßig Leer- und Tabulatorzeichen entfernt.

Rückgabewert

Typ: Zeichenkette

Diese Funktionen geben die getrimmte Version der angegebenen Zeichenkette zurück.

Beispiele

#1

text := "  Text  "
MsgBox
(
    "Ohne Trim:`t'" text "'
    Trim:`t'" Trim(text) "'
    LTrim:`t'" LTrim(text) "'
    RTrim:`t'" RTrim(text) "'"
)
MsgBox LTrim("00000123", "0")
