MenuFromHandle

Ruft ein Menu-Objekt ab, das zu einem Win32-Menü-Handle gehört.

Menu := MenuFromHandle(Handle)

Parameter

Handle

Der Handle eines Win32-Menüs (vom Typ HMENU).

Bemerkungen

Es wird nur das aktuelle Handle des Menüs erkannt. Wenn das vom Menü dazugehörige Win32-Menü zerstört und neu erstellt wird, ändert sich das Handle. Für weitere Details, siehe Win32-Menüs.

Wenn das Handle ungültig ist oder nicht zu einem Menü gehört, das von diesem Skript erstellt wurde, gibt die Funktion eine leere Zeichenkette zurück.
