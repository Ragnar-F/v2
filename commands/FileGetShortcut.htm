FileGetShortcut

Ermittelt Informationen über eine Verknüpfung (.lnk), wie z. B. ihre Zieldatei.

FileGetShortcut Verknüpfung , AusZiel, AusVerz, AusArgs, AusBeschreibung, AusSymbol, AusSymbolNr, AusMinMax

Parameter

Verknüpfung

Typ: Zeichenkette

Name der Verknüpfung, die untersucht werden soll. Wenn kein absoluter Pfad angegeben ist, wird die Verknüpfung im A_WorkingDir-Verzeichnis vermutet. Die Erweiterung .lnk muss mit angegeben werden.

AusZiel

Typ: Variable

Name der Variable, in der das Ziel der Verknüpfung gespeichert werden soll (ohne mögliche Parameter). Zum Beispiel: C:\WINDOWS\system32\notepad.exe

AusVerz

Typ: Variable

Name der Variable, in der das Arbeitsverzeichnis der Verknüpfung gespeichert werden soll. Zum Beispiel: C:\Meine Dokumente. Falls Umgebungsvariablen wie %WinDir% in der Zeichenkette vorhanden sind, können sie mit StrReplace aufgelöst werden. Zum Beispiel: AusVerz := StrReplace(AusVerz, "%WinDir%", A_WinDir)

AusArgs

Typ: Variable

Name der Variable, in der die Parameter der Verknüpfung gespeichert werden sollen (leer, wenn keine).

AusBeschreibung

Typ: Variable

Name der Variable, in der die Kommentare der Verknüpfung gespeichert werden sollen (leer, wenn keine).

AusSymbol

Typ: Variable

Name der Variable, in der der Dateiname des verwendeten Symbols gespeichert werden soll (leer, wenn keine).

AusSymbolNr

Typ: Variable

Name der Variable, in der die Nummer des verwendeten Symbols von einer Symboldatei gespeichert werden soll (leer, wenn keine). Dieser Wert ist meistens 1 (erstes Symbol).

AusMinMax

Typ: Variable

Name der Variable, in der der anfängliche Ausführungsstatus der Verknüpfung gespeichert werden soll. Das kann eine der folgenden Zahlen sein:

1: Normal
3: Maximiert
7: Minimiert

Fehlerbehandlung

Bei Misserfolg wird eine Ausnahme ausgelöst.

Bemerkungen

Man kann eine Ausgabevariable weglassen, wenn die entsprechende Information nicht benötigt wird.

Siehe auch

FileCreateShortcut, SplitPath

Beispiele

#1

Datei := FileSelect(32,, "Verknüpfung zur Überprüfung auswählen.", "Verknüpfungen (*.lnk)")
if Datei = ""
    return
FileGetShortcut Datei, AusZiel, AusVerz, AusArgs, AusBeschr, AusSymbol, AusSymbolNr, AusMinMax
MsgBox AusZiel "`n" AusVerz "`n" AusArgs "`n" AusBeschr "`n" AusSymbol "`n" AusSymbolNr "`n" AusMinMax
