ComObjActive

Ermittelt ein laufendes Objekt, das mit OLE registriert wurde.

ComObject := ComObjActive(CLSID)

Parameter

CLSID

Typ: Zeichenkette

CLSID oder eine visuell lesbare ProgID des COM-Objekts.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu einem laufenden COM-Objekt zurück.

Fehlerbehandlung

Wenn ComObjActive ein aktives Objekt nicht abrufen kann, wird die Funktion je nach aktueller ComObjError-Einstellung und anderen Faktoren eventuell eine Ausnahme auslösen, das Skript beenden oder eine leere Zeichenkette zurückgeben.

Siehe auch

ComObject, ComObjCreate, ComObjGet, ComObjConnect, ComObjError, ComObjFlags, ObjAddRef/ObjRelease, ComObjQuery, GetActiveObject (MSDN)

Beispiele

#1: Zeigt das aktive Dokument in Microsoft Word an, sofern das Programm läuft.

word := ComObjActive("Word.Application")
if !word
    MsgBox "Word ist nicht offen."
else
    MsgBox word.ActiveDocument.FullName
