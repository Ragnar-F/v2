Func

Ruft eine Referenz zu einer bestimmten Funktion ab.

Referenz := Func(Funktionsname)

Parameter

Funktionsname

Name einer Funktion, deren Referenz abgerufen werden soll. Funktionsname muss explizit im Skript definiert sein oder eine interne Funktion sein.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu Funktionsname zurück; sprich, ein Func- oder Closure-Objekt. Wenn Funktionsname keine interne Funktion ist oder im Skript nicht explizit vorhanden ist (mit Mitteln wie #Include oder einen nicht-dynamischen Aufruf einer Bibliotheksfunktion), wird eine leere Zeichenkette zurückgegeben.

Wenn Funktionsname eine verschachtelte Funktion innerhalb einer Funktion mit freien Variablen ist, wird ein neues Closure erzeugt und zurückgegeben. Es hat die gleichen Eigenschaften wie Func.

Bemerkungen

Das zurückgegebene Objekt kann genutzt werden, um die Funktion aufzurufen oder um Informationen wie die minimale und maximale Anzahl von Parametern abzurufen.

Beispiele

; Ruft eine Referenz zu der Funktion "StrLen" ab.
fn := Func("StrLen")

; Zeigt Informationen über die Funktion an.
MsgBox fn.Name "() ist " (fn.IsBuiltIn ? "integriert." : "benutzerdefiniert.")
