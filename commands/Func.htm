Func

Ruft eine Funktionsreferenz ab.

Referenz := Func(FunktionName)

Parameter

FunktionName

Typ: Zeichenkette

Name einer Funktion, deren Referenz abgerufen werden soll. FunktionName muss explizit im Skript definiert sein oder eine interne Funktion sein.

Rückgabewert

Typ: Objekt

Diese Funktion gibt eine Referenz zu FunktionName zurück; sprich, ein Func- oder Closure-Objekt. Wenn FunktionName keine interne Funktion ist oder im Skript nicht explizit vorhanden ist (mit Mitteln wie #Include oder einen nicht-dynamischen Aufruf einer Bibliotheksfunktion), wird eine leere Zeichenkette zurückgegeben.

Wenn FunktionName eine verschachtelte Funktion innerhalb einer Funktion mit freien Variablen ist, wird ein neues Closure erzeugt und zurückgegeben. Es hat die gleichen Eigenschaften wie Func.

Bemerkungen

Das zurückgegebene Objekt kann genutzt werden, um die Funktion aufzurufen oder um Informationen wie die minimale und maximale Anzahl von Parametern abzurufen.

Beispiele

#1: Ruft eine Funktionsreferenz ab und zeigt Informationen über diese Funktion an.

fn := Func("StrLen")
MsgBox fn.Name "() ist " (fn.IsBuiltIn ? "integriert." : "benutzerdefiniert.")
