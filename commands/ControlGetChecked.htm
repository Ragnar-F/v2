ControlGetChecked

Gibt einen Wert ungleich 0 zurück, wenn ein CheckBox- oder Radio-Steuerelement markiert ist.

IstMarkiert := ControlGetChecked(Steuerelement , Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Steuerelement

Typ: Zeichenkette, Integer oder Objekt

ClassNN, Text oder HWND des Steuerelements, oder ein Objekt mit einer Hwnd-Eigenschaft. Einzelheiten finden Sie unter Der Steuerelement-Parameter.

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie unter Fenstertitel-Parameter.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem internen Tool Window Spy aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Integer (boolescher Wert)

Diese Funktion gibt 1 (true) zurück, wenn das CheckBox- oder Radio-Steuerelement markiert ist, oder 0 (false), wenn nicht.

Fehlerbehandlung

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

Es wird eine Ausnahme ausgelöst, wenn ungültige Parameter erkannt werden.

Bemerkungen

Fenstertitel und -texte sind Groß-/Kleinschreibung-sensitiv. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
