String

Konvertiert einen Wert in eine Zeichenkette.

ZketteWert := String(Wert)

Rückgabewert

Das Ergebnis der Konvertierung von Wert in eine Zeichenkette, oder Wert selbst, wenn es eine Zeichenkette ist.

Wenn Wert eine Zahl ist, wird die Standard-Dezimalformatierung angewendet.

Wenn Wert ein Objekt ist, ist der Rückgabewert das Ergebnis des Aufrufs von Wert.ToString(). Wenn das Objekt keine solche Methode hat, wird eine Ausnahme ausgelöst. Diese Methode ist standardmäßig nicht implementiert und muss daher zuvor im Skript definiert werden.
