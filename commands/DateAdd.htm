DateAdd

Addiert oder Subtrahiert Zeit von einem Datum-Zeit-Wert.

AusgabeVar := DateAdd(DatumZeit, Zeit, Zeiteinheit)

Parameter

DatumZeit

Ein Zeitstempel im YYYYMMDDHH24MISS-Format.

Zeit

Die Zeitmenge, die addiert werden soll (Integer oder Gleitkommazahl). Mit einer negativen Zahl kann eine Subtraktion durchgefüht werden.

Zeiteinheit
Das Einheitszeichen des Zeit-Parameters. Zeiteinheit kann eine der folgenden Zeichenketten sein (oder jeweils nur erste Buchstabe davon):
Seconds, Minutes, Hours oder Days.

Rückgabewert

Enthält DatumZeit einen ungültigen Zeitstempel oder ein Jahr älter als 1601, oder ist Zeit nicht-numerisch, wird dieser Befehl eine leere Zeichenkette zurückgeben, um das Problem zu kennzeichnen.

Ansonsten wird ein Zeitstempel im YYYYMMDDHH24MISS-Format zurückgegeben.

Bemerkungen

Die Built-in-Variable A_Now enthält die aktuelle Ortszeit im YYYYMMDDHH24MISS-Format.

Mit DateDiff kann die Zeitmenge zwischen zwei Zeitstempeln berechnet werden.

Siehe auch

DateDiff, FileGetTime, FormatTime

Beispiel

; Addiert das heutige Datum um 31 Tage.
Zukunft := DateAdd(A_Now, 31, "days")
MsgBox % FormatTime(Zukunft)
