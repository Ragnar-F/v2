FileGetTime

Ermittelt den Zeitstempel einer Datei oder eines Ordners.

Zeitstempel := FileGetTime(Dateiname, WelcheZeit)

Parameter

Dateiname

Typ: Zeichenkette

Name der Zieldatei oder des Zielordners. Wenn kein absoluter Pfad angegeben ist, wird die Datei oder der Ordner im A_WorkingDir-Verzeichnis vermutet. Fehlt dieser Parameter, wird stattdessen die aktuelle Datei von der innersten umschließenden Dateimuster-Schleife verwendet.

WelcheZeit

Typ: Zeichenkette

Welcher Zeitstempel abgerufen werden soll:
M = Zeitpunkt der letzten Änderung (Standardwert, wenn dieser Parameter fehlt)
C = Zeitpunkt der Erstellung
A = Zeitpunkt des letzten Zugriffs

Rückgabewert

Typ: Zeichenkette

Diese Funktion gibt eine Zeichenkette von Ziffern im YYYYMMDDHH24MISS-Format zurück. Die Zeit ist die lokale Zeit des Betriebssystems, nicht UTC/GMT. Diese Zeichenkette sollte nicht als Zahl behandelt werden (man sollte keine mathematischen Operationen an ihr durchführen oder sie numerisch vergleichen).

ErrorLevel

ErrorLevel wird bei Misserfolg auf 1 und bei Erfolg auf 0 gesetzt.

A_LastError wird auf das Ergebnis der Systemfunktion GetLastError() gesetzt.

Bemerkungen

Siehe YYYYMMDDHH24MISS, um mehr über Datum- und Zeitangaben zu erfahren.

Siehe auch

FileSetTime, FormatTime, FileGetAttrib, FileSetAttrib, FileGetSize, FileGetVersion, Dateimuster-Schleife, DateAdd, DateDiff

Beispiel

Zeitstempel := FileGetTime("C:\Meine Dokumente\test.doc")  ; Ermittelt standardmäßig das Änderungsdatum.
Zeitstempel := FileGetTime("C:\Meine Dokumente\test.doc", "C")  ; Ermittelt das Erstellungsdatum.
