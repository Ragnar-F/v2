WinGetControlsHwnd

Gibt die eindeutigen ID-Nummern aller Steuerelemente in einem bestimmten Fenster zurück.

Steuerelemente := WinGetControlsHwnd(Fenstertitel, Fenstertext, Titelausnahme, Textausnahme)

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem mitgelieferten Windows Spy Tool aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Rückgabewert

Typ: Array

Diese Funktion gibt ein Array zurück, das die Window Handles (HWND) aller Steuerelemente im angegebenen Fenster enthält. Wenn keine passenden Fenster gefunden werden, wird eine leere Zeichenkette zurückgegeben.

Wenn zum Beispiel der Rückgabewert in eine Variable namens Steuerelemente gespeichert wird und zwei Steuerelemente vorhanden sind, enthält Steuerelemente[1] die ID des ersten Steuerelements, Steuerelemente[2] die ID des zweiten Steuerelements und Steuerelemente.Count() gibt die Zahl 2 zurück.

Steuerelemente werden nach ihrer Z-Reihenfolge sortiert, das üblicherweise der Navigation mit der Tab-Taste entspricht, sofern vom Fenster unterstützt.

Bemerkungen

Die ID des Fensters oder Steuerelements, das sich unter dem Mauszeiger befindet, kann mit MouseGetPos abgerufen werden.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.
