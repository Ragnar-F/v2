ToolTip

Erstellt ein Fenster irgendwo auf dem Bildschirm, das immer im Vordergrund ist.

ToolTip Text, X, Y, WelcherToolTip

Parameter

Text

Typ: Zeichenkette

Fehlt dieser Parameter oder ist er leer, wird der Tooltip versteckt, falls er bereits existiert. Ansonsten ist dieser Parameter der Text, der im Tooltip angezeigt werden soll. Um einen Tooltip mehrzeilig zu machen, trennt man jede Zeile mit einem Zeilenumbruchszeichen (`n), z. B. Zeile1`nZeile2.

Wenn der Text-Parameter relativ lang ist, kann er mithilfe eines Fortsetzungsbereichs in mehreren kurzen Zeilen aufgeteilt werden, um ihn übersichtlicher zu machen.

X, Y

Typ: Integer

Die X-Y-Koordinate der oberen linken Ecke des Tooltips, relativ zum aktiven Fenster (verwende CoordMode "ToolTip", wenn die Koordinate relativ zum Bildschirm sein soll). Fehlen die Koordinaten, wird der Tooltip in der Nähe des Mauszeigers angezeigt.

WelcherToolTip

Typ: Integer

Dieser Parameter dient dazu, mehrere Tooltips gleichzeitig anzeigen zu lassen. Eine Nummer zwischen 1 und 20 kennzeichnet, mit welchem ToolTip-Fenster agiert werden soll. Wenn keine Nummer angegeben ist, gilt standardmäßig 1 (das erste Fenster).

Bemerkungen

Ein typisches ToolTip sieht wie folgt aus:

Befindet sich die angegebene X-Y-Koordinate außerhalb des Bildschirms, wird der ToolTip neu positioniert, um vollständig sichtbar zu bleiben.

Der Tooltip wird solange angezeigt, bis eine der folgenden Bedingungen eintritt:

  • Das Skript wird beendet.
  • Die ToolTip-Funktion wird mit leerem Text-Parameter erneut ausgeführt.
  • Der Benutzer hat den Tooltip angeklickt (dieses Verhalten variiert je nach Betriebssystem).

Mithilfe der OwnDialogs-Option kann ein ToolTip einem GUI-Fenster untergeordnet werden. So ein untergeordneter ToolTip wird automatisch zerstört, sobald das Hauptfenster zerstört wird.

Beispiel

ToolTip "Mehrzeiliger`nTooltip", 100, 150

; Um einen ToolTip nach einer gewissen Zeit verschwinden zu lassen,
; ohne Sleep verwenden zu müssen (das den aktuellen Thread anhalten würde):
ToolTip "Zeitgesteuerter Tooltip`nDieser wird für 5 Sekunden angezeigt."
SetTimer () => ToolTip(), -5000
