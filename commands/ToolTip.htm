ToolTip

Erstellt ein Fenster irgendwo auf dem Bildschirm, das immer im Vordergrund ist.

ToolTip Text, X, Y, WelcherToolTip

Parameter

Text

Fehlt dieser Parameter oder ist er leer, wird der Tooltip versteckt, falls er bereits existiert. Ansonsten ist dieser Parameter der Text, der im Tooltip angezeigt werden soll. Um ein mehrzeiliges Tooltip zu erstellen, muss man zwischen jeder Zeile das Zeilenumbruchszeichen (`n) verwenden, z. B. Zeile1`nZeile2.

Wenn der Text zu lang ist, kann er in mehreren kurzen Zeilen aufgeteilt werden (Fortsetzungsbereich), um ihn besser lesbar und verwaltbar zu machen.

X, Y

Die X-Y-Koordinate der oberen linken Ecke des Tooltips, relativ zum aktiven Fenster (verwende CoordMode "ToolTip", wenn die Koordinate relativ zum Bildschirm sein soll). Fehlen die Koordinaten, wird der Tooltip in der Nähe des Mauszeigers angezeigt.

WelcherToolTip

Dieser Parameter dient dazu, mehrere Tooltips gleichzeitig anzeigen zu lassen. Eine Nummer zwischen 1 und 20 kennzeichnet, auf welchem Tooltip sich dieser Befehl beziehen soll. Fehlt dieser Parameter, gilt standardmäßig die Nummer 1 (das Erste).

Bemerkungen

Befindet sich die angegebene X-Y-Koordinate außerhalb des Bildschirms, wird der ToolTip neu positioniert, um vollständig sichtbar zu bleiben.

Der Tooltip wird solange angezeigt, bis eine der folgenden Bedingungen eintritt:

  • Das Skript wird beendet.
  • Die ToolTip-Funktion wird mit leerem Text-Parameter erneut ausgeführt.
  • Der Benutzer hat den Tooltip angeklickt (dieses Verhalten variiert je nach Betriebssystem).

Mithilfe der OwnDialogs-Option kann ein ToolTip an ein GUI-Fenster gebunden werden. So ein Tooltip wird automatisch zerstört, sobald das dazugehörige GUI-Fenster zerstört wird.

Beispiel

ToolTip "Mehrzeiliger`nTooltip", 100, 150

; Lässt einen Tooltip nach einer bestimmten Zeit verschwinden,
; ohne Sleep zu verwenden (das den aktuellen Thread anhalten würde):
ToolTip "Zeitgesteuerter Tooltip`nDieser wird für 5 Sekunden angezeigt."
SetTimer "ToolTipEntfernen", -5000
return

ToolTipEntfernen:
