MonitorGetCount

Gibt die Gesamtanzahl von Bildschirmen zurück.

Anzahl := MonitorGetCount()

Parameter

Diese Funktion hat keine Parameter.

Rückgabewert

Typ: Integer

Diese Funktion gibt die Gesamtanzahl von Bildschirmen zurück.

Bemerkungen

Im Gegensatz zur SM_CMONITORS-Systemeigenschaft, die via SysGet(80) abgerufen werden kann, umfasst der Rückgabewert alle Monitore, selbst solche, die nicht zum Desktop gehören.
