FileDelete

Löscht eine oder mehrere Dateien.

FileDelete Dateimuster

Parameter

Dateimuster

Name einer Datei oder eine musterbasierte Suche wie "C:\Temp\*.tmp". Dateimuster wird im A_WorkingDir-Verzeichnis vermutet, wenn kein absoluter Pfad angegeben ist.

Mit DirDelete kann ein ganzer Ordner gelöscht werden, einschließlich seine Unterordner und Dateien.

ErrorLevel

ErrorLevel wird bei Misserfolg auf die Anzahl der Dateien gesetzt, die nicht gelöscht werden konnten, und bei Erfolg auf 0. Das Löschen einer musterbasierten Suche wie "*.tmp" wird als Erfolg angesehen, auch dann, wenn keine Dateien gefunden wurden; folglich wird ErrorLevel auf 0 gesetzt.

Wenn Dateien gefunden wurden, wird A_LastError entweder auf Null (0) oder unmittelbar nach dem letzten Fehler auf das Ergebnis der Systemfunktion GetLastError() gesetzt. Ansonsten enthält A_LastError einen Fehlercode, der eventuell kenntlich macht, warum keine Dateien gefunden wurden.

Bemerkungen

Um eine schreibgeschützte Datei zu löschen, muss zuerst der Schreibschutzattribut entfernt werden. Zum Beispiel: FileSetAttrib "-R", "C:\Meine Datei.txt".

Beispiel
