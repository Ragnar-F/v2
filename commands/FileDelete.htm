FileDelete

Löscht eine oder mehrere Dateien.

FileDelete Dateimuster

Parameter

Dateimuster

Name einer Datei oder eine musterbasierte Suche wie "C:\Temp\*.tmp". Dateimuster wird im A_WorkingDir-Verzeichnis vermutet, wenn kein absoluter Pfad angegeben ist.

Mit DirDelete kann ein ganzer Ordner gelöscht werden, einschließlich seine Unterordner und Dateien.

ErrorLevel

ErrorLevel wird auf die Anzahl an Dateien gesetzt, die aufgrund eines Fehlers nicht gelöscht werden konnten, ansonsten auf 0. Das Löschen einer musterbasierten Suche wie "*.tmp" wird als Erfolg angesehen, auch dann, wenn keine Dateien gefunden wurden; folglich wird ErrorLevel auf 0 gesetzt.

Wenn Dateien gefunden wurden, wird A_LastError auf 0 gesetzt, oder auf das aktuellste Ergebnis der Systemfunktion GetLastError(). Ansonsten enthält A_LastError einen Fehlercode, der kenntlich machen soll, warum keine Dateien gefunden wurden.

Bemerkungen

Eine schreibgeschützte Datei kann nur gelöscht werden, wenn ihr vorher das Schreibschutzattribut entfernt wird. Zum Beispiel: FileSetAttrib "-R", "C:\Meine Datei.txt".
