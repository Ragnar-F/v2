WinMaximize

Vergrößert ein bestimmtes Fenster auf dessen maximale Größe.

WinMaximize Fenstertitel, Fenstertext, Titelausnahme, Textausnahme

Parameter

Fenstertitel

Typ: Zeichenkette

Der Titel oder andere Möglichkeiten zum Identifizieren des Zielfensters. Weitere Informationen finden Sie auf der Seite Fenstertitel-Parameter & Zuletzt Gefundenes Fenster.

Fenstertext

Typ: Zeichenkette

Eine Zeichenkette, die in einem der Textelemente des Zielfensters enthalten ist (eventuell vorhandene Textelemente können mit dem mitgelieferten Windows Spy Tool aufgedeckt werden). Versteckte Textelemente werden nur erkannt, wenn DetectHiddenText eingeschaltet ist.

Titelausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Titel diesen Wert enthalten.

Textausnahme

Typ: Zeichenkette

Ignoriert Fenster, deren Textelemente diesen Wert enthalten.

Bemerkungen

Mit WinRestore kann die Maximierung des Fensters aufgehoben werden. Mit WinMinimize kann es minimiert werden.

Wenn ein bestimmter Fenstertyp nicht richtig auf WinMaximize reagiert, könnte man stattdessen folgendes ausprobieren:

PostMessage 0x112, 0xF030,,, Fenstertitel, Fenstertext  ; 0x112 = WM_SYSCOMMAND, 0xF030 = SC_MAXIMIZE

Diese Funktion bezieht sich immer auf das oberste übereinstimmende Fenster; enthält Fenstertitel allerdings ahk_group Gruppenname, sind alle Fenster in der Gruppe betroffen.

Fenstertitel und -texte unterscheiden zwischen Groß- und Kleinschreibung. Versteckte Fenster werden nur erkannt, wenn DetectHiddenWindows eingeschaltet ist.

Siehe auch

WinRestore, WinMinimize

Beispiel

Run "notepad.exe"
WinWait "Unbenannt - Editor"
WinMaximize ; zuletzt gefundenes Fenster verwenden

^Up::WinMaximize "A"  ; Weist ein Hotkey zu, der das aktive Fenster maximieren soll.
