DirDelete

Löscht einen Ordner.

DirDelete VerzName , Rekursiv

Parameter

VerzName

Typ: Zeichenkette

Name des Verzeichnisses, das gelöscht werden soll. Wenn kein absoluter Pfad angegeben ist, wird das Verzeichnis im A_WorkingDir-Verzeichnis vermutet.

Rekursiv

Typ: Integer (boolescher Wert)

Dieser Parameter bestimmt, ob Unterverzeichnisse mit einbezogen werden sollen. Lässt man diesen Parameter weg, wird standardmäßig 0 (false) verwendet. Geben Sie einen der folgenden Werte an:

0 (false): Dateien und Unterverzeichnisse werden nicht aus VerzName entfernt. Wenn VerzName in diesem Fall nicht leer ist, wird nichts unternommen und eine Ausnahme ausgelöst.

1 (true): Entfernt alle Dateien und Unterverzeichnisse (wie der Windows-Befehl "rmdir /S").

Fehlerbehandlung

Es wird eine Ausnahme ausgelöst, wenn ein Fehler auftritt.

Bemerkungen

Keine.

Beispiele

#1

DirDelete "C:\Download-Temp"
DirDelete "C:\Download-Temp", 1
