GetMethod

Ruft die Implementierungsfunktion einer Methode ab.

Methode := GetMethod(Wert, Name)

Parameter

Wert

Typ: Any

Ein Wert beliebigen Typs, außer vom Typ ComObject.

Name

Typ: Zeichenkette

Der Name der Methode, die abgerufen werden soll.

Rückgabewert

Typ: Funktionsobjekt

Diese Funktion gibt das Funktionsobjekt, das die Implementierung der Methode enthält, zurück.

Fehler

Wenn die Methode nicht gefunden wurde, wird eine Ausnahme ausgelöst.

Bemerkungen

Die Methode kann vordefiniert (intern), innerhalb einer Klassendefinition oder durch Aufrufen von DefineMethod definiert sein. Geerbte Methoden werden in die Suche einbezogen. Methoden, die via __Call-Metafunktion implementiert wurden, können nicht erkannt werden und haben eventuell gar kein entsprechendes Funktionsobjekt.

Beim Aufruf des Funktionsobjekts ist es notwendig, einen Wert für den normalerweise versteckten this-Parameter zu übergeben. Zum Beispiel %Methode%(Wert).

Beispiele

#1

Methode := GetMethod({}, "GetMethod")  ; Sie ist auch eine Methode.
MsgBox Methode.MaxParams  ; Nimmt 2 Parameter, einschließlich 'this'.
MsgBox Methode = Func("GetMethod")  ; Tatsächlich das gleiche Objekt in diesem Fall.
