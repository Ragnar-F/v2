Enumerator-Objekt

Ein Enumerator ist eine Art Funktionsobjekt, das wiederholt aufgerufen wird, um eine Sequenz von Werten zu enumerieren.

Siehe auch: For-Schleife, OwnProps, OwnMethods, __Enum (Array), __Enum (Map)

Call

Ermittelt das nächste Element (auch mehrere) in einer Enumeration.

Enum.Call(ByRef AusgabeVar1 , ByRef AusgabeVar2)
AusgabeVar1, AusgabeVar2

Typ: Variable

Ermittelt einen umsetzungsspezifischen Wert.

Rückgabewert

Ein Integer ungleich Null bei Erfolg, oder Null, wenn keine Elemente übrig sind.
